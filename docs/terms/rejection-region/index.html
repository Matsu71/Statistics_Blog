まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

棄却域は、「ここまで外れたら帰無仮説では苦しい」と決めた領域です。

判定の境界線を図で見える形にしたものと考えると分かりやすいです。

中央に近い値は棄却域の外にあり、帰無仮説でも起こりうると考えます。

どちら側を棄却域にするかは、対立仮説の向きで決まります。

有意水準を小さくすると、棄却域はもっと端へ移ります。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 標準正規分布の端に色を塗り、その色付き部分を棄却域と考えると理解しやすいです。
  • 右片側検定では右端だけ、左片側検定では左端だけを塗ります。
  • 両側検定では左右両端を塗り、中央部分は棄却域の外になります。
  • 観測された検定統計量が色付きの部分に入るかどうかが判定になります。

定義

直感のあとで、言葉としての定義をここで固定します。

棄却域とは、帰無仮説 H0H_0H0​ が正しいとしたときに十分起こりにくい検定統計量の値の範囲です。観測された検定統計量がこの範囲に入ったら、帰無仮説を棄却します。棄却域は、有意水準 α\alphaα と対立仮説の向きによって決まります。右片側検定なら右端、左片側検定なら左端、両側検定なら両端に置かれるのが基本です。棄却域そのものは確率ではなく、「どの値なら棄却するか」という範囲の名前です。

現実でどう現れるか

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

棄却域は、検定統計量の値の範囲です。

棄却域は有意水準と対立仮説の向きから決まります。

観測値が棄却域に入れば帰無仮説を棄却します。

片側検定と両側検定で棄却域の位置は変わります。

p値で判定する方法と棄却域で判定する方法は、同じ判断を別の形で表しています。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

右片側検定の棄却域

右片側検定では、標準化した統計量 zzz が上側の臨界値 z1−αz_{1-\alpha}z1−α​ を超えたときに帰無仮説を棄却します。

R={z∣z>z1−α}R=\{z \mid z>z_{1-\alpha}\}R={z∣z>z1−α​}

記号の見方

  • RRR は棄却域です。
  • z1−αz_{1-\alpha}z1−α​ は標準正規分布の累積確率 1−α1-\alpha1−α に対応する値です。
  • z 検定での代表的な形です。

定義の式

両側検定の棄却域

両側検定では、正負どちら側でも十分大きな絶対値になったときに棄却します。有意水準は左右の尾に分かれます。

R={z∣∣z∣>z1−α/2}R=\{z \mid |z|>z_{1-\alpha/2}\}R={z∣∣z∣>z1−α/2​}

記号の見方

  • 左右それぞれに α/2\alpha/2α/2 ずつ割り当てます。
  • 観測値の符号ではなく絶対値の大きさで極端さを見ます。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

片側検定と両側検定で同じ棄却域を使うこと

生データの値の範囲と検定統計量の範囲を混同すること

観測後に有意水準や棄却域を動かしてしまうこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

棄却域は、帰無仮説のもとでの検定統計量の分布から作られます。したがって、「どの値なら珍しいか」を数値で表したものとみなせます。

有意水準 α\alphaα を固定したら、棄却域はその確率が α\alphaα 以下になるように選ばれます。これが第一種の過誤をコントロールする仕組みです。

観測値が棄却域に入るかどうかを見る方法と、p値が有意水準以下かを見る方法は同値です。ただし、前者は臨界値比較、後者は確率比較という見方の違いがあります。

棄却域は生データの範囲ではなく、通常は検定統計量の範囲として定義されます。ここを混同しないことが大切です。

証明の流れを見る 棄却域は、帰無仮説のもとで起こりにくい尾の部分を集めた領域として作られます。
  1. まず、帰無仮説 H0H_0H0​ のもとで検定統計量 TTT の分布を決めます。
  2. 次に、有意水準 α\alphaα に応じて、その分布の端から合計で α\alphaα 以下の確率になる部分を選びます。
  3. その部分を棄却域 RRR とすると、TTT が RRR に入ったときは H0H_0H0​ のもとで珍しい結果だと判断できます。
  4. したがって、棄却域は「帰無仮説からどれだけ離れたら棄却するか」を分布上で示した領域だと分かります。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

null-hypothesis を基準にすると、棄却域は「その仮説のもとで起こりにくい範囲」として理解できます。どの仮説の分布に対して棄却域を作っているのかを見失わないことが大切です。

p-value は、その棄却域の考え方を確率に言い換えたものです。臨界値と比べるか、p値と有意水準を比べるかは違っても、最終的な判定は同じになります。
