まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

右上がりの傾向と右下がりの傾向が全体として打ち消し合うと、相関係数は 0 になります。

点が丸くばらばらに散っている散布図では、無相関に近そうだと感じられます。

しかし U 字型のように、関係はあるのに直線では表しにくい形でも無相関になることがあります。

無相関は「予測不可能」と同じ意味でもありません。直線だけではうまく表せないことがあります。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 点が中心のまわりにほぼ円形に散っていれば、右上がり・右下がりの偏りが小さく、無相関に近いです。
  • 左右対称な U 字型では、右側は右上がり、左側は右下がりなので、全体として直線傾向が消えることがあります。
  • 無相関の散布図でも、帯ではなく曲線や塊が見えるなら、関係を「なし」と決めつけてはいけません。

定義

直感のあとで、言葉としての定義をここで固定します。

無相関とは、2 つの変量の相関係数が 0 であり、直線的な意味での結びつきが見られない状態です。平均的に見て、一方が大きいほど他方も大きい、あるいは小さいといった直線的傾向が打ち消し合っています。ただし無相関は「どんな関係もない」という意味ではなく、曲線的な関係や特別な構造が残っていることがあります。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

相関係数 r=0r=0r=0 の意味を問う問題

直線的な関係が弱いデータを、散布図と合わせて解釈するとき

独立と無相関の違いを整理するとき

U字型の売上

気温が低すぎても高すぎても売上が落ち、中間温度で売上が高い場合、散布図は U 字型になります。このとき相関係数は 0 に近くても、温度と売上に関係がないわけではありません。

完全にばらばらな 2 変量

サイコロの出目と、その日の通し番号のように対応関係がほぼない組では、散布図に明確な傾きが見えず、無相関とみなされやすいです。

独立との違い

2 つの変量が独立なら無相関になることがありますが、無相関だから必ず独立とは限りません。試験ではこの向きの違いを区別して読みます。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

無相関は、相関係数が 0 であることを指します。

無相関は「直線的な関係がない」という意味であり、「どんな関係もない」とは限りません。

散布図を見て、曲線的な関係が隠れていないか確認することが大切です。

無相関と独立は同じ意味ではありません。

相関係数だけでなく、関係の形を確認する姿勢が重要です。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

無相関の条件

相関係数が 0 であるとき、その 2 変量は無相関といいます。ここで言っているのは直線的な関係の有無です。

r=0r=0r=0

記号の見方

  • rrr は相関係数です。
  • 曲線的な関係の有無までは、この式だけでは分かりません。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

無相関を、完全な無関係と同じ意味だと思うこと

無相関なら独立だと決めつけること

相関係数が 0 に近いだけで、散布図を確認しないこと

U 字型のような曲線関係を見落とすこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

相関係数は直線的な結びつきの強さを測るので、r=0r=0r=0 で分かるのは直線的傾向が打ち消し合っていることまでです。

たとえば Y=X2Y=X^2Y=X2 のように左右対称な関係では、XXX の正負で傾きの向きが変わるため、全体では相関が 0 になることがあります。

したがって、無相関から「関数関係はない」「独立である」とまでは言えません。相関係数の守備範囲を超えた読み方をしないことが大切です。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

相関係数2つの変量の直線的な関係の向きと強さを、-1 から 1 の範囲で表す指標です。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

correlation-coefficient のページで相関係数を 1 つの数として読めるようになったら、その次に大事なのが無相関の意味です。無相関は「直線では説明しにくい」という情報であって、「関係が何もない」と言い切る言葉ではありません。
