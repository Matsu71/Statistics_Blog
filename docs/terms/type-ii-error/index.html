  • 実際には差や効果があるのに、帰無仮説を棄却できず見逃してしまう誤りです。
  • 効果があるのに検出できなかった状況を説明する場面
  • 第二種の過誤は「差があるのに棄却できない」誤りです。

まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

第二種の過誤は、本当は異常があるのに警報を鳴らし損ねる状態です。

標本が小さい、ばらつきが大きい、差が小さいときに起こりやすくなります。

帰無仮説を棄却できなかったからといって、差がないと確定したわけではありません。

有意水準を厳しくすると、第一種の過誤は減りやすい一方で、第二種の過誤は増えやすくなります。

第二種の過誤の裏返しが検出力です。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 対立仮説が真のときの分布を描き、その上で棄却域の外側に残る面積が第二種の過誤に関係します。
  • 真の差が小さいほど、その分布は帰無仮説の分布と重なりやすく、見逃しやすくなります。
  • 標本サイズを大きくすると分布のぶれが小さくなり、棄却域に届きやすくなります。
  • 第一種の過誤は帰無仮説側の誤り、第二種の過誤は対立仮説側の誤りと見ると整理しやすいです。

定義

直感のあとで、言葉としての定義をここで固定します。

第二種の過誤とは、実際には対立仮説の側に差や効果があるのに、検定の結果として帰無仮説 H0H_0H0​ を棄却できない誤りです。いわば「差があるのに差がないと判断してしまう」誤りで、偽陰性や false negative にあたります。第二種の過誤確率は通常 β\betaβ と書かれ、標本サイズ、ばらつき、真の差の大きさ、有意水準などで変わります。第二種の過誤が小さいほど、真の差を見つけやすくなります。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

効果があるのに検出できなかった状況を説明する場面

検出力や標本サイズ設計の基本を考える場面

第一種の過誤とのトレードオフを問う場面

「棄却できない」を「正しい」と誤読しないための確認場面

効果がある薬を見逃す

実際には薬が効いているのに、標本サイズが小さくばらつきも大きいため p値が大きくなり、帰無仮説を棄却できなかったとします。これは第二種の過誤です。

小さな改善を拾えない

工程改善で平均重量がわずかに変わっていても、標本数が少ないと検定統計量が十分大きくならず、異常を見逃すことがあります。差が小さいほど第二種の過誤は起こりやすくなります。

標本サイズを増やす効果

同じ差でも標本サイズを増やすと標準誤差が小さくなり、検定統計量は大きくなりやすくなります。その結果、第二種の過誤は小さくなりやすいです。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

第二種の過誤は「差があるのに棄却できない」誤りです。

第二種の過誤確率は通常 β\betaβ で表します。

第二種の過誤は、標本サイズ、ばらつき、真の差の大きさ、有意水準に影響されます。

検出力は 1−β1-\beta1−β で、真の差を見つける力を表します。

帰無仮説を棄却できないことと、帰無仮説が真であることは同じではありません。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

第二種の過誤確率

実際には対立仮説の側にあるのに、検定統計量 TTT が棄却域 RRR に入らず、帰無仮説を棄却できない確率が第二種の過誤確率です。

β=P(T∉R∣H1)\beta=P(T\notin R\mid H_1)β=P(T∈/R∣H1​)

記号の見方

  • RRR は棄却域です。
  • H1H_1H1​ は対立仮説が真である状況を表します。
  • 複数の真の値を含む対立仮説では、β\betaβ はどの真の値かによって変わります。

定義の式

検出力

真の差があるときに、それを正しく検出して帰無仮説を棄却できる確率を検出力といいます。

Power=1−β\mathrm{Power}=1-\betaPower=1−β

記号の見方

  • 第二種の過誤を小さくするほど、検出力は大きくなります。
  • 標本サイズ設計では、必要な検出力を先に決めることがあります。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

第二種の過誤を「帰無仮説が真なのに棄却する誤り」と取り違えること

帰無仮説を棄却できなかったら、帰無仮説が真だと断定すること

第二種の過誤は常に 1−α1-\alpha1−α だと思うこと

真の差の大きさや標本サイズが第二種の過誤に影響しないと思うこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

第二種の過誤は、対立仮説が真である側での見逃しです。したがって、その大きさは「どれくらいの差が本当にあるか」に依存します。

同じ検定でも、真の差が大きければ検定統計量は棄却域へ届きやすくなり、第二種の過誤は小さくなります。差が小さいと見逃しやすくなります。

標本サイズを大きくすると標準誤差が小さくなり、同じ差でも検出しやすくなるため、第二種の過誤は一般に小さくなりやすいです。

第一種の過誤を抑えるために有意水準を下げすぎると、第二種の過誤は大きくなりやすいので、両者は一緒に考える必要があります。

証明の流れを見る 第二種の過誤は、対立仮説が真のときに観測統計量が棄却域へ届かないことで起こります。
  1. まず、真の値が対立仮説の側にある状況を考えます。
  2. そのときの検定統計量 TTT の分布を考えると、帰無仮説のときより分布の中心がずれることがあります。
  3. しかし、ばらつきや標本サイズの影響で、観測値が棄却域 RRR に届かないこともあります。
  4. その確率 P(T∉R∣H1)P(T\notin R\mid H_1)P(T∈/R∣H1​) が第二種の過誤確率 β\betaβ です。
  5. したがって、第二種の過誤は「真の差があるのに検定で拾えない確率」だと分かります。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

第一種の過誤帰無仮説が本当は正しいのに、それを棄却してしまう誤りです。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

type-i-error と比べると、第二種の過誤は誤りの向きが逆です。第一種の過誤が「ない差をあると見てしまう誤り」なら、第二種の過誤は「ある差を見逃す誤り」です。

この違いを押さえると、検出力の意味も見えやすくなります。帰無仮説を棄却できなかった場面では、差がなかったのか、差があったのに検出力が足りなかったのかを区別して考える必要があります。
