まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

ポアソン分布は、「一定の範囲で何回起きるか」を数える分布です。

1 時間あたりの電話件数、1 ページあたりの誤字数のような回数データで使われます。

平均回数 λ\lambdaλ が分布の中心と広がりを決めます。

二項分布で試行回数が多く、成功確率が小さい場面の近似として出ることもあります。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 横軸に 0 回、1 回、2 回と発生回数を並べ、各回数の確率を棒で表します。
  • λ\lambdaλ が小さいと 0 回や 1 回に確率が集まりやすくなります。
  • λ\lambdaλ が大きくなると、山は右へ移動します。

定義

直感のあとで、言葉としての定義をここで固定します。

ポアソン分布は、一定の時間や空間の中で、まれな出来事が何回起こるかを表す離散分布です。平均発生回数を λ\lambdaλ として表します。

平均と分散が同じ分布の例として

電話の件数

コールセンターに 1 分あたり平均 3 件の電話が来るなら、1 分間に何件来るかをポアソン分布で近似できることがあります。

誤字の数

1 ページあたりの誤字数のように、一定範囲での発生回数を見るときに使われます。

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

パラメータ λ\lambdaλ は平均発生回数です。

期待値と分散はどちらも λ\lambdaλ です。

二項分布との関係が問われることがあります。

ポアソン分布は、非負整数を取る離散分布です。

平均発生回数 λ\lambdaλ が大きいほど、分布の山は大きな回数側へ動きます。

二項分布で nnn が大きく ppp が小さく、npnpnp が一定程度のときの近似として使われることがあります。
