まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

ベイズの定理は、「原因から結果」だけでなく「結果から原因」を考える道具です。

検査で陽性だったとき、実際に病気である確率を考えるような場面で使います。

条件を逆にすると分母が変わるため、直感だけでは間違えやすいところを式で整理します。

事前にどれくらい起こりやすいかと、観測結果がどれくらい起こりやすいかを合わせて考えます。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 全体を原因 AAA と AcA^cAc に分け、それぞれから結果 BBB が起こる枝を考えます。
  • BBB が起きた後は、BBB に到達した枝だけを集めて、その中で AAA から来た割合を見ます。
  • 樹形図で分母と分子を整理すると、条件の向きの違いが見えやすくなります。

定義

直感のあとで、言葉としての定義をここで固定します。

ベイズの定理は、P(A∣B)P(A\mid B)P(A∣B) と P(B∣A)P(B\mid A)P(B∣A) を結びつける公式です。ある結果 BBB が分かったあとで、その原因や仮説 AAA の確率を更新して考えるときに使います。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

検査の陽性・陰性のように、結果から原因を考える問題

条件付き確率の向きを入れ替える問題

事前確率と事後確率を区別したい場面

検査結果の読み取り

病気の人が少ない集団では、検査が陽性でも実際に病気である確率は直感より小さくなることがあります。ベイズの定理は分母に陽性全体を置いて整理します。

原因を逆向きに考える

雨の日に傘を持つ人が多いとしても、傘を持っている人が必ず雨を経験したとは限りません。条件の向きを入れ替えると意味が変わります。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

P(A∣B)P(A\mid B)P(A∣B) と P(B∣A)P(B\mid A)P(B∣A) を取り違えないことが重要です。

分母は、結果 BBB が起こる全体の確率です。

樹形図で枝の確率をかけ、同じ結果に到達する経路を足すと整理しやすいです。

事前確率、尤度、事後確率の役割を区別します。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

ベイズの定理

結果 BBB が起きたもとで、原因や仮説 AAA が成り立つ確率を表します。

P(A∣B)=P(B∣A)P(A)P(B)P(A\mid B)=\frac{P(B\mid A)P(A)}{P(B)}P(A∣B)=P(B)P(B∣A)P(A)​

記号の見方

  • P(B)>0P(B)>0P(B)>0 が必要です。
  • P(A)P(A)P(A) は事前の起こりやすさです。

定義の式

分母の分解

事象 AAA とその補事象で場合分けして、BBB が起こる全体の確率を求めます。

P(B)=P(B∣A)P(A)+P(B∣Ac)P(Ac)P(B)=P(B\mid A)P(A)+P(B\mid A^c)P(A^c)P(B)=P(B∣A)P(A)+P(B∣Ac)P(Ac)

記号の見方

  • AcA^cAc は AAA の補事象です。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

P(A∣B)P(A\mid B)P(A∣B) と P(B∣A)P(B\mid A)P(B∣A) を同じだと思うこと

分母に P(A)P(A)P(A) を置いてしまうこと

事前確率を無視して検査の精度だけで判断すること

補事象から BBB が起こる経路を足し忘れること

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

ベイズの定理は、条件付き確率の定義 P(A∣B)=P(A∩B)/P(B)P(A\mid B)=P(A\cap B)/P(B)P(A∣B)=P(A∩B)/P(B) と P(B∣A)=P(A∩B)/P(A)P(B\mid A)=P(A\cap B)/P(A)P(B∣A)=P(A∩B)/P(A) から導けます。

分母 P(B)P(B)P(B) は、観測された結果が起こる全ての経路を足したものです。

条件の向きを入れ替えると、分母が変わるため値も一般には変わります。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

確率事象が起こる見込みを 0 から 1 の数で表したものです。条件付き確率ある事象が起きたという条件のもとで、別の事象が起こる確率です。
