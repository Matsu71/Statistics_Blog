正規分布

この記事でわかること

  • 左右対称のつり鐘型をした、もっとも代表的な連続型の確率分布です。
  • 測定誤差や自然なばらつきを考える場面で、近似モデルとしてよく使われます。
  • 正規分布は平均を中心に左右対称です。

まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

平均の近くの値はよく現れ、平均から遠い値は現れにくいという形を、なめらかな曲線で表したものです。

左右が対称なので、平均より少し大きい値と少し小さい値は同じくらい起こりやすいと考えます。

身長や測定誤差のように、多くは真ん中付近に集まり、極端な値は少ない場面をイメージすると入りやすいです。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 真ん中がいちばん高く、左右に行くほどなめらかに低くなるつり鐘型の曲線として表されます。
  • 曲線の下の面積が確率を表すので、中央付近の面積が大きいほど、その範囲の値が出やすいと読めます。

定義

直感のあとで、言葉としての定義をここで固定します。

確率変数が平均の近くに集まり、平均から離れるほど起こりにくくなる連続型の確率分布を正規分布といいます。平均と分散によって形と広がりが決まります。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

  • 測定誤差や自然なばらつきを考える場面で、近似モデルとしてよく使われます。
  • 標本平均の分布、区間推定、仮説検定、標準化など、統計検定2級の重要分野に何度も登場します。
  • 標準正規分布や z 値の理解に進む前の土台になります。

身長や測定値のばらつき

多くの人の身長や、同じ製品を繰り返し測った値は、平均付近に集まり、極端に大きい値や小さい値は少なくなることがあります。そうした形を近似するのに正規分布が使われます。

推定と検定の入り口

標本平均がどのくらいぶれるかを考えるときや、ある値が平均から見てどれほど離れているかを評価するときに、正規分布の考え方が使われます。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

  • 正規分布は平均を中心に左右対称です。
  • 平均と分散が分かると、正規分布の位置と広がりが決まります。
  • 標準化によって標準正規分布へ変換する流れは頻出です。
  • 平均の近くにデータが集まりやすく、遠い値ほど起こりにくいという読み方を押さえます。
  • 平均から 1 標準偏差、2 標準偏差、3 標準偏差の範囲には、それぞれおよそ 68%、95%、99.7% が入る近似がよく使われます。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

正規分布の確率密度関数

f(x) = 1 / (√(2π)σ) · exp(-(x - μ)^2 / (2σ^2))

平均 μ の近くで値が大きく、そこから離れるほど小さくなる形を表す式です。曲線の下の面積が確率を表します。

  • μ は平均です。
  • σ は標準偏差で、σ > 0 です。
  • x は連続的に変わる値です。

標準化

Z = (X - μ) / σ

正規分布に従う確率変数 X を、平均 0、分散 1 の標準正規分布へ移す基本変換です。

  • X は平均 μ、標準偏差 σ の正規分布に従うとします。
  • 単位の違う値同士でも、平均からどれくらい離れているかを比較しやすくなります。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

  • つり鐘型に見えれば何でも正規分布だと考えてしまうこと
  • 連続型の分布なのに、1 点にちょうど一致する確率をそのまま読もうとすること
  • 正規分布の広がりを表す分散や標準偏差を無視して、平均だけで形が決まると思ってしまうこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

正規分布は連続型の分布なので、1 点の確率を直接考えるのではなく、ある区間に入る確率を曲線下の面積として考えます。

平均 μ は分布の中心を決め、分散 σ^2 は広がりを決めます。分散が大きいほど曲線は横に広がり、頂点は低くなります。

標準化により、異なる平均や分散を持つ正規分布を同じ基準で比較できるようになります。

証明の流れを見る 標準化の式に従って変数変換を行うと、一般の正規分布を標準正規分布に移せることが確かめられます。
  1. X から平均 μ を引いて中心を 0 に移します。
  2. 標準偏差 σ で割って広がりを 1 にそろえます。
  3. こうして得られる Z を標準正規分布で扱えるようになります。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

平均データ全体の中心を手早くつかむための、もっとも基本的な代表値です。分散データが平均のまわりにどのくらい散らばっているかを、二乗を使って測る代表的な指標です。標準偏差分散を元の単位に戻して解釈しやすくした、散らばりの代表的な指標です。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

正規分布は、単独の分布の話に見えて、実際には推定や検定へつながる土台の役割を持っています。形だけを覚えるのではなく、「平均が中心を決め、分散が広がりを決める」「標準化で共通のものさしに移せる」という 2 点まで押さえておくと先の学習がかなり楽になります。
