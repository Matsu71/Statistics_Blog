まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

ベルヌーイ分布は、はい/いいえ、成功/失敗を 1 回だけ見る分布です。

表が出たら 1、裏が出たら 0 のように、結果を 0 と 1 にします。

二項分布は、このベルヌーイ試行を何回も独立に繰り返した成功回数の分布です。

確率 ppp が大きいほど、1 が出やすくなります。

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 横軸に 0 と 1 だけを置き、それぞれに 1−p1-p1−p と ppp の棒を立てます。
  • p=0.5p=0.5p=0.5 なら 0 と 1 の棒は同じ高さです。
  • ppp が 1 に近づくほど、1 の棒が高くなります。

定義

直感のあとで、言葉としての定義をここで固定します。

ベルヌーイ分布は、成功を 1、失敗を 0 とする 1 回の試行を表す分布です。成功確率を ppp とすると、P(X=1)=pP(X=1)=pP(X=1)=p、P(X=0)=1−pP(X=0)=1-pP(X=0)=1−p です。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

ベルヌーイ分布と二項分布を同じものだと思うこと

成功確率と失敗確率を足して 1 になることを忘れること

0/1 以外の値をそのままベルヌーイ分布として扱うこと

独立な繰り返しの前提を確認しないこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

ベルヌーイ分布は、0 と 1 の 2 値だけを取る離散分布です。

期待値は 0⋅(1−p)+1⋅p=p0\cdot(1-p)+1\cdot p=p0⋅(1−p)+1⋅p=p です。

独立なベルヌーイ試行を複数回足し合わせると、成功回数は二項分布に従います。
