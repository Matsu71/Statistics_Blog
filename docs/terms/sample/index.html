標本

母集団から取り出して、実際に観測や集計の対象にした一部のデータです。

まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

全体を全部調べるのが難しいときに、一部だけ取り出して様子を見るのが標本です。

標本は、母集団の代わりに先に観察する小さなデータの集まりです。

まず標本を見てから、その結果をもとに母集団の傾向を考えます。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 100 人の集団全体を母集団、そのうち色を変えて取り出した 10 人を標本と考えると、全体と一部の違いが見やすくなります。
  • 母集団から標本を取り出し、集計して、そこから全体を考えるという「母集団 → 標本 → 集計 → 推測」の流れで捉えると整理しやすいです。

定義

直感のあとで、言葉としての定義をここで固定します。

標本とは、母集団全体を直接調べるのが難しいときに、母集団から取り出して実際に観測した一部のデータをいいます。たとえば、全校生徒の身長を知りたいが 50 人だけを測ったなら、その 50 人分の身長データが標本です。統計では、この標本を使って母集団の特徴を考えます。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

  • 標本調査で、母集団から実際にどの対象を調べたかを整理する場面
  • アンケートや製品検査のように、全体ではなく一部を調べて判断する場面
  • 標本平均や標本分散など、標本から計算する量を導入する場面

学年 40 人の先行集計

学年 40 人のテスト結果全体を知りたいが、先に 8 人分だけ集めて 58, 62, 64, 67, 70, 72, 75, 76 を得たとします。この 8 人分が標本で、ここから全体の傾向を考えます。

菓子袋の内容量検査

1 日に作った菓子袋全部の内容量を知りたいとき、30 袋だけ抜き取って重さを測れば、その 30 袋が標本です。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

  • 標本は母集団そのものではなく、母集団から取り出した一部です。
  • 「知りたい全体」が母集団で、「実際に調べた一部」が標本です。
  • 手元にあるデータ全部が、そのまま母集団とは限りません。
  • 標本から計算した量と、母集団に対して定まる量は区別して考えます。
  • 標本は、標本平均のような統計量を作る出発点です。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

標本の表し方

x_1, x_2, \ldots, x_n

標本は、観測された値を x_1, x_2, ..., x_n のように並べて表せます。

  • x_i は i 番目の観測値です。
  • n は標本サイズです。
  • N を母集団の大きさとして区別して考えます。

標本平均

\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i

標本に含まれる観測値の平均が標本平均です。

  • x_i は i 番目の観測値です。
  • n は標本サイズです。
  • 標本から計算する統計量の例です。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

  • 手元にあるデータ全部を、そのまま母集団だと思ってしまうこと
  • 標本そのものと標本平均を同じものとして扱うこと
  • 標本サイズ n と母集団の大きさ N を混同すること
  • 標本なら自動的に母集団をよく表していると考えてしまうこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

標本は、母集団から得られた一部の観測値です。標本を x_1, x_2, ..., x_n と書くとき、n は標本サイズであり、母集団の大きさ N とは区別します。

標本平均や標本分散は、標本から計算する統計量です。これらは母平均や母分散そのものではなく、母集団を考えるための手がかりです。

手元のデータが全部そろっていても、それが結論を述べたい全体でなければ母集団ではなく標本として扱います。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

母集団統計で知りたい対象全体を指し、標本がどこから取り出されたかを決める出発点です。標本平均標本に含まれる観測値を合計して個数で割った、標本を代表する平均です。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

標本は、population である母集団から実際に取り出して観測した一部です。統計では、この標本を出発点にして全体の様子を考えます。

標本に含まれる観測値から平均を計算したものが sample-mean です。つまり、標本は「調べた一部」、標本平均は「その一部から計算した代表値」と分けて捉えると整理しやすいです。
