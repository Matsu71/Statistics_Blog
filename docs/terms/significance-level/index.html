まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

有意水準は、「どれくらい厳しく棄却するか」を先に決めるルールです。

0.05 なら、帰無仮説のもとでめったに起きない 5% 程度の領域を棄却域にします。

0.01 に下げると、もっと珍しい結果だけを棄却の対象にします。

有意水準は標本ごとに変わる値ではなく、検定設計の値です。

厳しくしすぎると、今度は本当にある差も見逃しやすくなります。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 帰無仮説の分布の端に、全体で 5% や 1% の面積を塗って棄却域を作る図を思い浮かべると分かりやすいです。
  • 有意水準が小さいほど、塗られる領域は分布のさらに外側へ移ります。
  • 片側検定ならその面積を片側に、両側検定なら左右に分けて置きます。
  • p値は観測後に計算する面積で、有意水準は先に決める面積という違いがあります。

定義

直感のあとで、言葉としての定義をここで固定します。

有意水準とは、仮説検定で帰無仮説 H0H_0H0​ が正しいのに棄却してしまう危険をどこまで許すかを、あらかじめ決めておく基準値です。通常は α\alphaα と書き、0.05 や 0.01 がよく使われます。有意水準はデータを見る前に設定するもので、観測後に計算する p値とは役割が異なります。有意水準を小さくすると棄却しにくくなり、第一種の過誤を起こしにくくなる一方で、差を見逃しやすくなることがあります。

0.05 と 0.01 の違い

同じデータで p値が 0.03 でも、有意水準 0.05 なら棄却し、有意水準 0.01 なら棄却しません。有意水準が厳しいほど、棄却のハードルは高くなります。

品質管理の警報基準

工場で異常を見逃したくないが、誤警報も多すぎると困るとき、どこで警報を出すかを先に決める感覚が有意水準に近いです。検査結果が出てから基準を動かしてはいけません。

片側と両側で使い方が違う

有意水準 5% でも、右片側検定では右側 5% を全部使い、両側検定では左右 2.5% ずつに分けます。同じ 5% でも棄却域の形は変わります。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

有意水準 α\alphaα は、データを見る前に決める基準です。

有意水準は p値ではなく、p値と比較される側の値です。

有意水準を小さくすると棄却しにくくなり、第一種の過誤を抑えやすくなります。

同じ有意水準でも、片側検定と両側検定で棄却域の位置は変わります。

第一種の過誤を抑えるほど、第二種の過誤が増えやすいことがあります。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

有意水準の基本条件

棄却域 RRR は、帰無仮説が正しいときに検定統計量 TTT が入り込む確率が有意水準 α\alphaα 以下になるように決めます。

PH0(T∈R)≤αP_{H_0}(T\in R)\le \alphaPH0​​(T∈R)≤α

記号の見方

  • RRR は棄却域です。
  • TTT は検定統計量です。
  • 離散分布の検定などでは、ちょうど α\alphaα ではなく「以下」になることがあります。

定義の式

p値による判定規則

観測データから計算した p値 ppp が有意水準 α\alphaα 以下なら、帰無仮説を棄却します。

p≤α⇒reject H0p\le \alpha \Rightarrow \text{reject } H_0p≤α⇒reject H0​

記号の見方

  • ppp は観測後に計算する p値です。
  • α\alphaα は観測前に決めておく有意水準です。
  • 片側検定か両側検定かに応じて p値の計算方法は変わります。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

有意水準を「帰無仮説が真である確率」だと思うこと

有意水準と p値を同じものとして扱うこと

標本を見たあとで有意水準を都合よく変えること

有意水準を小さくすれば、いつでも検定が良くなると思うこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

有意水準は、検定手続きそのものに属する設計値です。したがって、同じ検定手順を繰り返したときに第一種の過誤がどれくらい起こりうるかを制御する役割を持ちます。

連続分布を使う基本的な検定では、棄却域の確率をちょうど α\alphaα に合わせることが多いですが、離散分布では「α\alphaα 以下」にしかできないことがあります。

有意水準を小さくすると、帰無仮説を棄却するために必要な証拠はより強くなります。その代わり、真の差があるときにも棄却できない第二種の過誤が増えやすくなります。

有意水準は研究や業務の目的に応じて決めますが、標本を見たあとに変更すると検定の意味が崩れます。

証明の流れを見る 有意水準は、帰無仮説のもとで棄却域に入る確率を上から抑えるように、観測前に設定されます。
  1. まず、帰無仮説のもとでの検定統計量 TTT の分布を考えます。
  2. その分布の端から、合計で α\alphaα 以下の確率になるように棄却域 RRR を決めます。
  3. 観測後は、実際の統計量が RRR に入ったか、または p値が α\alphaα 以下かで判定します。
  4. したがって、有意水準は観測後に変わる値ではなく、観測前に固定する判定基準だと分かります。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

p値帰無仮説が正しいとしたときに、観測結果と同じかそれ以上に極端な結果が出る確率です。棄却域帰無仮説のもとで十分に起こりにくいとみなした検定統計量の範囲で、そこに入ったら帰無仮説を棄却します。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

p-value と対で理解すると、有意水準は「先に決める基準」、p値は「あとで計算する結果」だとはっきりします。ここを取り違えないことが、仮説検定の読み間違いを減らす近道です。

rejection-region は、この有意水準を図の上で表したものです。有意水準が決まると、どの範囲を棄却域にするかが決まり、そこに観測値が入るかどうかで判定できます。
