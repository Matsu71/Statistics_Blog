分散

データが平均のまわりにどのくらい散らばっているかを、二乗を使って測る代表的な指標です。

この記事でわかること

  • データが平均のまわりにどのくらい散らばっているかを、二乗を使って測る代表的な指標です。
  • 平均だけでは見えないデータの広がり方を比べたいときに使います。
  • 分散は散らばりの大きさを見る基本指標です。

まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

平均が同じ 2 つのデータでも、値が平均の近くに集まる場合と大きく離れる場合があります。

分散は、その「平均からの離れやすさ」をひとまとめにした数字です。

偏差をそのまま足すと正負が打ち消し合うので、二乗してから平均するという工夫をしています。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 平均が同じでも、点が狭く集まる集団は分散が小さく、広く散る集団は分散が大きくなります。
  • 平均から遠い値ほど二乗の効果で強く効くため、外れた値があると分散は大きくなりやすいです。

定義

直感のあとで、言葉としての定義をここで固定します。

各データの平均からのずれを二乗し、その平均をとった値を分散といいます。平均が同じでも、広がり方が違うデータを区別するために使います。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

  • 平均だけでは見えないデータの広がり方を比べたいときに使います。
  • 標準偏差、正規分布、推定、検定などを学ぶ前提として繰り返し登場します。
  • 品質管理やテスト結果の安定性の比較のように、ばらつき自体が重要な場面で現れます。

テストの得点分布

平均点がどちらも 70 点でも、60 点から 80 点に集中しているクラスと、30 点から 100 点まで広がっているクラスでは分散が違います。後者のほうが分散は大きいです。

工場での測定値

製品の長さや重さが平均値の近くで安定しているかを見るときに、分散や標準偏差が使われます。安定していれば分散は小さくなります。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

  • 分散は散らばりの大きさを見る基本指標です。
  • 分散の単位は元データの単位ではなく、その二乗になります。
  • 標準偏差は分散の平方根であり、単位を元に戻した指標です。
  • 分散は平均からのずれを二乗しているので、外れ値の影響を受けやすいです。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

母分散

σ^2 = (1 / n) Σ (x_i - x̄)^2

平均からのずれを二乗し、その平均をとると分散になります。データ全体の広がりを 1 つの値にまとめた形です。

  • x_i は i 番目の観測値です。
  • x̄ は平均です。
  • 平均から遠い値ほど、二乗によって寄与が大きくなります。

分散と標準偏差の関係

σ = √(σ^2)

分散の平方根をとると標準偏差になります。単位を元のデータに戻して解釈しやすくしたものです。

  • 分散と標準偏差は役割が近いですが、単位が異なります。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

  • 分散の値を、元データと同じ単位でそのまま解釈してしまうこと
  • 分散が大きいことを、平均が大きいことと混同すること
  • 標準偏差と分散を同じものとして扱い、平方根の関係を説明できないこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

偏差の和 Σ(x_i - x̄) は 0 になるので、散らばりの大きさをそのまま測るには使いにくいです。そのため、負の値も正にし、遠い値を強く反映する二乗を使います。

分散は「平均からの二乗距離の平均」とみることができます。この見方を持つと、標本分散や不偏分散の違いも整理しやすくなります。

証明の流れを見る 偏差の和が 0 になることを確認すると、なぜ単純な偏差の平均では散らばりを測れないかが分かります。
  1. 偏差の和 Σ(x_i - x̄) を考えます。
  2. x̄ = (1 / n) Σx_i を代入して式を整理します。
  3. 偏差の和が 0 になるので、散らばりを見るには二乗など別の工夫が必要だと分かります。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

平均データ全体の中心を手早くつかむための、もっとも基本的な代表値です。標準偏差分散を元の単位に戻して解釈しやすくした、散らばりの代表的な指標です。正規分布左右対称のつり鐘型をした、もっとも代表的な連続型の確率分布です。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

分散は、平均だけでは見えない「広がり方」を言葉と式の両方でつかむための入口です。ここで二乗する理由まで納得しておくと、標準偏差や正規分布、さらに推定や検定へ進んだときに理解がつながりやすくなります。
