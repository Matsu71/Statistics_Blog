まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

標準化したずれを二乗して足し合わせた量だと考えると、カイ二乗分布の意味がつかみやすいです。

二乗しているので負の値は出ません。

自由度が大きくなるほど、二乗して足す項が増え、分布の中心は右へ動きます。

各項が「平均との差の大きさ」を表しているので、全体としてずれが大きいほどカイ二乗値も大きくなります。

カイ二乗分布は正規分布とは違い、左右対称ではありません。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 横軸の 0 から始まり、右に裾を引く分布として描かれます。
  • 自由度 1 や 2 では強く右に歪みますが、自由度が増えると山がなだらかになっていきます。
  • いくつもの二乗したずれを足し合わせた面積や大きさを見ている、と考えるとイメージしやすいです。

定義

直感のあとで、言葉としての定義をここで固定します。

カイ二乗分布は、独立な標準正規変数 Z1,…,ZkZ_1,\dots,Z_kZ1​,…,Zk​ の二乗和 Z12+⋯+Zk2Z_1^2+\cdots+Z_k^2Z12​+⋯+Zk2​ で定義される確率分布で、自由度 kkk を持ちます。二乗和なので値は必ず 0 以上になり、自由度が小さいと右に長い裾を持つ形になります。正規母集団の分散に関する推定や検定、分割表の検定統計量などで重要な役割を持ちます。

数式とその意味

定義の式

カイ二乗分布の定義

独立な標準正規変数の二乗和が、自由度 kkk のカイ二乗分布です。

X=∑i=1kZi2∼χ2(k)X=\sum_{i=1}^{k}Z_i^2 \sim \chi^2(k)X=i=1∑k​Zi2​∼χ2(k)

  • kkk が自由度です。

定義の式

平均と分散

自由度 kkk のカイ二乗分布の中心と広がりを表す基本性質です。

E[X]=k,Var⁡(X)=2kE[X]=k,\quad \operatorname{Var}(X)=2kE[X]=k,Var(X)=2k

記号の見方

  • X∼χ2(k)X\sim\chi^2(k)X∼χ2(k) のときに成り立ちます。

定義の式

標本分散との関係

正規母集団からの標本では、標本分散を母分散で標準化した統計量がカイ二乗分布に従います。分散の推定や検定の土台になる式です。

(n−1)S2σ2∼χ2(n−1)\frac{(n-1)S^2}{\sigma^2}\sim \chi^2(n-1)σ2(n−1)S2​∼χ2(n−1)

記号の見方

  • 正規母集団からの標本であることを前提にします。
  • S2S^2S2 は不偏分散、σ2\sigma^2σ2 は母分散です。ここでは手元のデータの分散とは区別します。

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

カイ二乗分布は、標準正規変数の二乗和で定義されるので、値は必ず 0 以上です。この性質は分散のような「大きさ」を扱う量と相性がよいです。

自由度は、独立な標準正規変数をいくつ足したかを表します。自由度が増えると、分布の中心は大きくなります。

正規母集団の標本分散にカイ二乗分布が現れるため、分散に関する区間推定や検定ではこの分布が重要になります。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

f-distribution は、独立な 2 つのカイ二乗分布を自由度で割って比にした分布です。したがって、カイ二乗分布で「二乗したずれを足し合わせる」「自由度で形が変わる」という感覚をつかんでおくと、F 分布の意味も追いやすくなります。

カイ二乗分布は、平均のまわりのずれを二乗して集める場面でよく現れます。分散や検定統計量に出てくるのは、その量が 0 以上で、ずれの大きさをまとめているからです。
