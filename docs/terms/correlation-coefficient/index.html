30秒で要点

この記事でわかることを短く確認します。

  • 2つの変量の直線的な関係の向きと強さを、-1 から 1 の範囲で表す指標です。
  • 散布図を 1 つの数で要約して比較したいとき
  • 相関係数の値は −1-1−1 から 111 の範囲をとります。

まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

共分散は単位で大きさが変わりますが、相関係数は標準化しているので比較しやすいです。

点が細く右上がりに並ぶ散布図なら、相関係数は 1 に近づきます。

点が細く右下がりに並ぶ散布図なら、相関係数は -1 に近づきます。

点が広く散っていれば、相関係数は 0 に近づきます。

相関係数は「関係の向きと直線的な強さ」を要約する道具で、予測は回帰直線の仕事です。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 同じ右上がりでも、点が直線の近くに密集しているほど相関係数の絶対値は大きくなります。
  • 点が右上がりでも扇形に広がるなら、正の相関はあっても強くはありません。
  • U 字型の散布図では関係がありそうでも、直線的ではないため相関係数は 0 に近いことがあります。

定義

直感のあとで、言葉としての定義をここで固定します。

相関係数は、2 つの変量がどれくらい直線的に一緒に増減するかを、単位の影響を取り除いて表す指標です。値は −1-1−1 から 111 の間をとり、111 に近いほど強い正の直線関係、−1-1−1 に近いほど強い負の直線関係、0 に近いほど直線関係が弱いと読みます。ただし、相関係数は関係を説明的に要約する量であり、片方からもう片方を予測する式そのものではありません。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

散布図を 1 つの数で要約して比較したいとき

2 つの量の直線的な結びつきを簡潔に述べたいとき

回帰直線や決定係数の理解に進む前に、説明的な関係の強さを整理するとき

勉強時間と得点

散布図が右上がりで点も比較的まとまっていれば、相関係数は正で大きめになります。これは勉強時間が長いほど得点が高い傾向を要約したもので、何点上がるかを直接予測する式ではありません。

身長と体重

身長と体重には正の相関が見られやすいですが、相関が高くても個人差は残ります。相関係数は「平均的に一緒に増えやすい」ことを示すだけで、各人の体重をぴったり当てるものではありません。

U字型の失敗例

温度と売上が中間で最大になる U 字型の関係では、散布図では関係が見えるのに相関係数は 0 に近くなることがあります。相関係数は直線的な関係だけを見ています。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

相関係数の値は −1-1−1 から 111 の範囲をとります。

符号は関係の向き、絶対値は直線的な関係の強さの目安です。

相関係数が大きくても、因果関係があるとは限りません。

相関係数が 0 に近くても、直線でない関係が隠れていることがあります。

相関係数は説明的な要約であり、予測に使うのは回帰直線です。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

相関係数

共分散 sxys_{xy}sxy​ を、xxx と yyy の標準偏差 sxs_xsx​、sys_ysy​ で割って標準化したものが相関係数です。単位の違いを取り除いて、直線的な関係の強さを比べやすくします。

r=sxysxsyr=\frac{s_{xy}}{s_x s_y}r=sx​sy​sxy​​

記号の見方

  • sxys_{xy}sxy​ は xxx と yyy の共分散です。
  • sxs_xsx​ と sys_ysy​ はそれぞれの標準偏差です。
  • sx>0s_x>0sx​>0、sy>0s_y>0sy​>0 を前提にします。

定義の式

相関係数の範囲

相関係数は必ず −1-1−1 以上 111 以下になります。端に近いほど直線的な並びが強く、0 に近いほど直線関係は弱いと読めます。

−1≤r≤1-1 \le r \le 1−1≤r≤1

記号の見方

  • これは直線的な関係の強さについての範囲です。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

相関係数を、因果関係の強さだと思うこと

相関係数が高いなら、そのまま精度の高い予測ができると考えること

相関係数が 0 なら、どんな関係もないと決めつけること

散布図を見ずに、相関係数の数値だけで判断すること

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

相関係数は、共分散を標準偏差で割って無次元化した量です。そのため、cm と kg のように単位が違う変量同士でも比較しやすくなります。

rrr は xxx と yyy を対称に扱う量であり、xxx と yyy を入れ替えても値は変わりません。この点は、どちらからどちらを予測するかで式が変わる回帰と異なります。

相関係数が表すのは直線的な関係の強さです。曲線的な関係や外れ値の影響は、散布図も合わせて確認しないと見誤ることがあります。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

無相関相関係数が 0 で、2つの変量に直線的な関係が見られない状態です。回帰直線説明変数 $x$ から目的変数 $y$ を予測するときに、平均的な関係を直線で表したものです。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

相関係数は、散布図の形を「直線的な関係の強さ」という 1 つの数に要約したものです。regression-line はここからさらに進んで、説明変数 xxx を決めたうえで目的変数 yyy を予測する直線なので、対称な相関係数とは役割が違います。no-correlation もあわせて見ると、r=0r=0r=0 の読み方を誤解しにくくなります。
