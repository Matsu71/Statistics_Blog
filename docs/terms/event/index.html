Term

事象

標本空間の中で、確率を考えたい結果のまとまりです。

重要ポイント

  • 事象は標本空間の部分集合です。
  • 和事象は「少なくともどちらか」、積事象は「両方」です。
  • 補事象は、標本空間の中でその事象が起こらない部分です。

定義

この用語が指しているものを、先に言葉で固定します。

事象は、標本空間の部分集合です。サイコロで偶数が出る事象なら、標本空間 {1,2,3,4,5,6} の中の {2,4,6} です。

例

直感、図での見方、現れる場面をまとめて確認します。

直感

事象は、「この条件を満たしたら起きたと言う」と決める問いです。

標本空間が全体の箱なら、事象はその中で注目して色を付けた部分です。

1 つの結果だけでなく、複数の結果をまとめて 1 つの事象にできます。

確率は、事象がどれくらい起こりやすいかを表します。

よく出てくる場面

確率を分子と分母で計算する基本問題

和事象、積事象、補事象を考えるとき

条件付き確率や独立の前提を整理するとき

偶数が出る事象

サイコロで偶数が出る事象は {2,4,6} です。標本空間のうち、条件を満たす結果だけを集めます。

補事象を見る

「偶数が出る」の補事象は「偶数が出ない」、つまり {1,3,5} です。起きない側を見ると確率計算が楽になることがあります。

視覚的な見方

  • サイコロの標本空間 {1,2,3,4,5,6} のうち、偶数の事象は {2,4,6} です。
  • 「3 以上が出る」は {3,4,5,6} で、別の事象です。
  • 同じ標本空間の中で、目的に合わせて注目する部分を変えるのが事象です。

試験でのポイント

問題文で見分けるための着眼点を整理します。

事象は標本空間の部分集合です。

和事象は「少なくともどちらか」、積事象は「両方」です。

補事象は、標本空間の中でその事象が起こらない部分です。

条件付き確率では、どの事象を条件としているかを確認します。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

事象

事象 AAA は標本空間 Ω\OmegaΩ の部分集合として表せます。

A⊆ΩA\subseteq\OmegaA⊆Ω

記号の見方

  • AAA は事象です。
  • Ω\OmegaΩ は標本空間です。

定義の式

補事象

事象 AAA が起こらない結果全体を補事象と呼びます。

Ac=Ω∖AA^c=\Omega\setminus AAc=Ω∖A

記号の見方

  • AcA^cAc は AAA の補事象です。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

事象を 1 つの結果だけだと思うこと

標本空間を確認せずに事象だけを考えること

和事象と積事象を取り違えること

補事象を「反対の言葉」だけで考え、集合として確認しないこと

詳しい説明

前提や証明の流れは、必要になったときに開いて確認できます。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

事象は集合として扱うため、和、積、補集合の考え方が使えます。

確率の計算では、どの結果がその事象に含まれるかを明確にすることが重要です。

条件付き確率では、条件となる事象の中に標本空間を狭めて考えます。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

確率事象が起こる見込みを 0 から 1 の数で表したものです。条件付き確率ある事象が起きたという条件のもとで、別の事象が起こる確率です。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

event は sample-space の一部です。probability では、その事象が標本空間の中でどれくらい起こりやすいかを数で表します。conditional-probability では、ある事象が起きたという条件のもとで別の事象を見ます。
