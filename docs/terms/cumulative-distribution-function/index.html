まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

累積分布関数は、左から順に確率を足し上げたものです。

xxx が大きくなるほど、X≤xX\le xX≤x に含まれる範囲が広がるため、値は小さくなりません。

最終的には全体の確率 1 に近づきます。

累積分布関数は、離散型にも連続型にも使える分布の表現です。

連続型では密度の面積を左から積み上げたものとして読めます。
