まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

検定統計量は、たくさんの標本情報を判定しやすい 1 つの尺度にまとめたものです。

同じ差でも、ばらつきが小さいと意味が大きく、ばらつきが大きいと意味が小さくなります。

そのため、差だけでなく「どれくらいぶれやすいか」で割って比べる考え方がよく使われます。

観測値が帰無仮説の分布の中央に近ければ、特別に珍しいとは言いにくいです。

観測値が分布の端に離れるほど、帰無仮説と合いにくいと考えます。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 帰無仮説のもとで中心が 0 の分布を描き、その上に観測された検定統計量を置く図を考えると役割がつかみやすいです。
  • 0 に近い値は「帰無仮説と大きくはずれていない」ことを表しやすいです。
  • 正の大きな値や負の大きな値は、基準からのずれが大きいことを表します。
  • 片側検定か両側検定かで、どの端を極端とみなすかが変わります。

定義

直感のあとで、言葉としての定義をここで固定します。

検定統計量とは、標本データから計算して、帰無仮説 H0H_0H0​ とどれくらい食い違っているかを 1 つの数で表したものです。平均の差や比率の差をそのまま使うのではなく、ばらつきや標本サイズも考慮して標準化することが多いです。こうして作った量を、帰無仮説のもとでの分布と比べて、p値や棄却域による判定につなげます。検定統計量はデータを要約した材料であり、p値そのものではありません。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

母平均や母比率の検定で、標本の差を標準化して判定するとき

z 検定や t 検定の計算問題で、まず値を求める場面

p値や棄却域を求める前段階として、標本を 1 つの尺度へ要約する場面

検定結果を「差の大きさ」と「ぶれやすさ」の両方から読む場面

平均の差を標準化する

基準値 100、母標準偏差 8、標本サイズ 16 のとき、標本平均が 104 なら Z=(104−100)/(8/16)=2Z=(104-100)/(8/\sqrt{16})=2Z=(104−100)/(8/16​)=2 です。平均との差 4 だけでなく、平均のぶれの大きさ 2 と比べて 2 倍離れていると読めます。

ばらつきが大きいと同じ差でも弱くなる

同じく平均との差が 4 でも、母標準偏差が 16 なら Z=(104−100)/(16/16)=1Z=(104-100)/(16/\sqrt{16})=1Z=(104−100)/(16/16​)=1 です。同じ差でも、ばらつきが大きいほど検定統計量は小さくなります。

p値への入口

観測された検定統計量が 2 なら、その値以上に極端な値が帰無仮説のもとでどれくらい出るかを調べて p値を計算します。検定統計量は p値を求めるための出発点です。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

検定統計量は、標本から計算する判定用の統計量です。

差の大きさだけでなく、ばらつきや標本サイズも反映した形が多いです。

観測された検定統計量から p値を計算したり、棄却域に入るかを判定したりします。

検定統計量そのものと、その値から得られる p値は別物です。

片側検定では符号の向きが特に重要になります。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

平均の差や比率の差そのものを、検定統計量と同じだと思うこと

検定統計量と p値を同じものとして扱うこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

検定統計量は、帰無仮説のもとで分布が分かるように設計されます。その分布が分かるからこそ、観測値がどれくらい珍しいかを確率で表せます。

差をそのまま使わず標準誤差で割るのは、標本サイズやばらつきが違う問題でも、極端さを同じ尺度で比較しやすくするためです。

記号 TTT は確率変数としての検定統計量を表し、実際に標本から得た数値は tobst_{\mathrm{obs}}tobs​ や zobsz_{\mathrm{obs}}zobs​ と書き分けることがあります。試験では、この区別が p値の説明で重要になります。

片側検定では、統計量の符号がどちら向きの差かを示します。両側検定では、絶対値の大きさで極端さを見ることが多いです。

証明の流れを見る 検定統計量は、差を標準誤差で割ることで「どれだけ極端か」を比較しやすい尺度に変えています。
  1. まず、標本から得た差 θ^−θ0\hat{\theta}-\theta_0θ^−θ0​ を考えます。
  2. この差だけでは、ばらつきが大きい標本と小さい標本を同じ基準で比べられません。
  3. そこで、差を標準誤差 SE(θ^)\mathrm{SE}(\hat{\theta})SE(θ^) で割って、ぶれの大きさをそろえます。
  4. 標準化した量 TTT は、帰無仮説のもとでの分布と比べやすくなり、p値や棄却域の判定に使えるようになります。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

p値帰無仮説が正しいとしたときに、観測結果と同じかそれ以上に極端な結果が出る確率です。棄却域帰無仮説のもとで十分に起こりにくいとみなした検定統計量の範囲で、そこに入ったら帰無仮説を棄却します。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

p-value は、観測された検定統計量がどれくらい珍しいかを確率に直したものです。つまり、検定統計量は標本を 1 つの数に要約する段階、p値はその数を確率で読む段階と分けて理解すると整理しやすくなります。

rejection-region は、その検定統計量に対して「どこまで行ったら棄却か」を決めた領域です。計算した値が棄却域に入るかどうかを見る方法と、p値で比べる方法は同じ判断を別の形で表しています。
