まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

最頻値は、データの中で「いちばんよく出る値」を見る指標です。

平均が合計、中央値が順位を見るのに対し、最頻値は出現回数を見ます。

アンケートで最も多く選ばれた選択肢のように、よくあるパターンを知りたいときに役立ちます。

データが山を 2 つ持つような形なら、最頻値が 2 つ現れることがあります。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 1, 2, 2, 3, 4, 4, 4 では、4 が 3 回出るので最頻値は 4 です。
  • 棒グラフで見ると、最も高い棒に対応する値が最頻値です。
  • 階級に分けたヒストグラムでは、最も度数が大きい階級を最頻階級として読むことがあります。

定義

直感のあとで、言葉としての定義をここで固定します。

最頻値は、データの中で出現回数が最も多い値です。平均や中央値のように 1 つに決まるとは限らず、複数ある場合や、はっきりした最頻値がない場合もあります。カテゴリデータにも使いやすい代表値です。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

アンケートで最も多い回答を知りたいとき

サイズ、色、カテゴリのように平均を取りにくいデータを要約するとき

ヒストグラムの山の位置をざっくり読みたいとき

最頻値を求める

2, 3, 3, 4, 5, 5, 5 の最頻値は 5 です。最も多く現れる値を数えるだけなので、平均を計算できないカテゴリでも使えます。

複数の最頻値

1, 1, 2, 3, 3 では 1 と 3 が同じ回数だけ現れます。このように最頻値が 2 つ以上になることがあります。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

最頻値は出現回数が最大の値です。

複数の最頻値がある場合や、どの値も同じ回数で出る場合があります。

平均・中央値・最頻値は、同じデータでも異なる値になることがあります。

カテゴリデータにも使える代表値として押さえます。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

最頻値は必ず 1 つだけだと思い込むこと

平均や中央値と同じ値になるはずだと考えること

度数を確認せず、値の大きさだけで最頻値を選ぶこと

最頻値だけでデータのばらつきまで分かると思うこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

最頻値は、データの中心というよりも「出現頻度の山」を表す代表値です。

数値データだけでなく、カテゴリデータでも使えます。

データ数が少ない場合は、少し値が増減するだけで最頻値が変わることがあります。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

平均データ全体の中心を手早くつかむための、もっとも基本的な代表値です。中央値データを小さい順に並べたとき、真ん中にくる値を表す代表値です。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

最頻値は、平均や中央値とあわせて代表値の違いを整理するための用語です。mean は合計、median は順位、mode は出現回数を見ると考えると、問題文の意図を読み分けやすくなります。
