まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

条件付き確率は、見ている全体を条件の中に絞り込む考え方です。

「全員の中で」ではなく「条件を満たした人の中で」と母集団を狭めます。

条件を付けると、同じ事象でも確率が変わることがあります。

独立かどうかを調べるときにも、条件付き確率が基準になります。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 標本空間全体のうち、まず事象 BBB の領域だけを残します。
  • その BBB の中で、AAA と重なっている部分がどれくらいあるかを見ます。
  • AAA と BBB の重なりが大きいほど、P(A∣B)P(A\mid B)P(A∣B) は大きくなります。

定義

直感のあとで、言葉としての定義をここで固定します。

条件付き確率は、事象 BBB が起きたと分かっている状況で、事象 AAA が起こる確率です。標本空間を BBB の中に狭めてから、AAA がどれくらい含まれるかを見ます。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

「すでに分かっている条件」がある確率問題

独立の定義を確認するとき

ベイズの定理で条件を入れ替えるとき

カードを引く例

赤いカードを引いたと分かっているとき、そのカードがハートである確率を考えるなら、全カードではなく赤いカードの中だけで割合を見ます。

条件で全体が変わる

ある検査で陽性だった人の中で実際に病気である確率は、全員の中で病気である確率とは別です。条件によって見ている集団が変わります。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

条件付き確率は、条件となる事象の中に標本空間を狭めて考えます。

P(A∣B)P(A\mid B)P(A∣B) と P(B∣A)P(B\mid A)P(B∣A) は一般には同じではありません。

分母は条件となる事象の確率です。

独立なら P(A∣B)=P(A)P(A\mid B)=P(A)P(A∣B)=P(A) になります。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

条件付き確率

事象 BBB が起きたもとで、AAA も起きる確率を表します。

記号の見方

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

P(A∣B)P(A\mid B)P(A∣B) と P(B∣A)P(B\mid A)P(B∣A) を同じだと思うこと

分母を P(A)P(A)P(A) にしてしまうこと

条件を付けたのに標本空間全体で考え続けること

条件付き確率と積事象の確率を混同すること

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

条件付き確率は、確率の基準となる全体を BBB に取り替える操作です。

P(A∣B)P(A\mid B)P(A∣B) は AAA と BBB の重なりを BBB の大きさで割ったものです。

条件を逆にすると分母が変わるため、P(A∣B)P(A\mid B)P(A∣B) と P(B∣A)P(B\mid A)P(B∣A) を区別します。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

独立一方の事象が起きても、もう一方の起こりやすさが変わらない関係です。ベイズの定理条件付き確率の向きを入れ替えて考えるための公式です。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

conditional-probability は、条件が付いたときに確率の分母が変わることを学ぶページです。independence では、条件を付けても確率が変わらない場合を扱います。bayes-theorem は条件付き確率を組み替える公式です。
