まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

残差は、回帰直線で予測したあとに「まだ残ったずれ」です。

実際の点が直線より上なら正、下なら負になります。

残差が全体として小さければ、回帰直線はデータによくあてはまっていると考えられます。

残差が規則的な形で並ぶなら、直線モデルが関係の形を取り逃しているかもしれません。

残差は回帰の量であって、相関係数のような対称な要約とは役割が違います。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 散布図の各点から回帰直線に下ろした縦の線分の長さが残差の大きさです。
  • 点が直線のすぐ近くに並ぶほど、残差は小さくなります。
  • 残差が上側にばかり続いたり、波のような形を描いたりするなら、単純な直線では十分でない可能性があります。

定義

直感のあとで、言葉としての定義をここで固定します。

残差は、各観測について実際の値 yiy_iyi​ から回帰直線による予測値 y^i\hat{y}_iy^​i​ を引いた ei=yi−y^ie_i=y_i-\hat{y}_iei​=yi​−y^​i​ のことです。点が回帰直線より上にあれば残差は正、下にあれば負になります。残差は標本データに対して計算される観測後のずれであり、理論モデルに出てくる誤差項そのものと同一ではありません。回帰のあてはまりを判断するときは、残差の大きさや並び方を見ます。

売上の予測誤差

ある日の売上の予測値が 100 個で、実際の売上が 110 個なら残差は 110−100=10110-100=10110−100=10 です。予測より実際が上に外れたことを意味します。

試験で重要なポイント

点が回帰直線より上なら残差は正、下なら負です。

最小二乗法は残差の二乗和を最小にする方法です。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

残差

実際の観測値から回帰直線の予測値を引いたものが残差です。回帰直線で説明しきれなかった部分を表します。

ei=yi−y^ie_i=y_i-\hat{y}_iei​=yi​−y^​i​

  • yiy_iyi​ は観測値、y^i\hat{y}_iy^​i​ は予測値です。
  • iii は各観測を表します。

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

残差は、標本から推定した回帰直線に対する観測後のずれです。したがって、母集団モデルに出てくる誤差項 ε\varepsilonε と概念的には近いですが、同一ではありません。

単回帰では xxx から yyy を予測するので、残差は縦方向のずれとして測ります。ここでも向きが決まっている点が、相関との違いです。

最小二乗法では、個々の残差ではなく残差平方和を小さくします。そのため、平均的に小さなずれだけでなく、大きな外れも重く評価されます。

残差の大きさだけでなく並び方も重要で、規則的なパターンがあれば直線モデルの見直しが必要なことがあります。

証明の流れを見る 切片を含む最小二乗回帰では、正規方程式から残差の総和が 0 になることが分かります。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

最小二乗法回帰直線を引くときに、観測値と予測値のずれの二乗和が最小になるよう係数を決める方法です。決定係数切片を含む単回帰で、回帰直線が目的変数の変動をどの程度とらえられているかを 0 から 1 の範囲で表す指標です。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

least-squares-method は、この residual の二乗和を最小にするように回帰直線を決める方法です。さらに coefficient-of-determination は、残差として残った変動が全体の変動に対してどれくらい小さいかを割合で見ます。

残差は回帰の文脈で初めて意味を持つ量なので、相関係数のような対称な関係の要約とは別物です。相関が高いかどうかだけでは、どの点がどれだけ外れているかは分からないため、残差を見る価値があります。
