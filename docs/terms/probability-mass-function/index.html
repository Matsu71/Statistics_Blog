まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

確率質量関数は、離散型の分布を棒の高さで表すものです。

サイコロの 1, 2, 3, 4, 5, 6 にそれぞれ確率を置くイメージです。

確率は各点に乗っていて、すべて足すと 1 になります。

連続型の確率密度関数とは、1 点の確率を直接扱うかどうかが違います。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 横軸に値、縦軸にその値を取る確率を置く棒グラフを考えます。
  • 棒の高さをすべて足すと 1 になります。
  • 二項分布やベルヌーイ分布は、確率質量関数で表せます。

定義

直感のあとで、言葉としての定義をここで固定します。

確率質量関数は、離散型確率変数 XXX が値 xxx を取る確率 P(X=x)P(X=x)P(X=x) を表す関数です。値ごとの確率を直接並べて見るための道具です。
