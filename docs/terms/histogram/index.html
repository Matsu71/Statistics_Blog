まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

ヒストグラムは、数値データを幅ごとの箱に分けて、どの箱に何個入ったかを見る図です。

棒が高いところは、その範囲にデータが多いことを表します。

データの中心だけでなく、分布の形や左右の偏りを見やすくします。

箱ひげ図が要約を短く見せる図なら、ヒストグラムは分布の形をより直接見せる図です。

ただし、階級にまとめるため、元の個別の値そのものは図から完全には戻せません。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 0 点台、10 点台、20 点台のように階級を作り、それぞれに何人いるかを柱で表します。
  • 柱が中央付近に集まれば中心に多い分布、片側に長く伸びれば偏りのある分布として読めます。
  • 階級幅を変えると見え方も変わるため、問題文で指定された階級をよく確認します。

定義

直感のあとで、言葉としての定義をここで固定します。

ヒストグラムは、連続的な数値データをいくつかの階級に分け、それぞれの階級に入るデータ数を柱で表した図です。分布の山、偏り、散らばり、外れた値の有無を視覚的に確認できます。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

テスト点数、身長、待ち時間など、数値データの分布を見るとき

正規分布に近い形か、左右に偏っているかをざっくり確認するとき

箱ひげ図と合わせて、分布の形と要約を比較するとき

点数分布を見る

テスト点数を 0-9 点、10-19 点、20-29 点のように分けると、どの点数帯に人数が多いかを柱の高さで見られます。

階級幅で見え方が変わる

5 点刻みと 20 点刻みでは、同じデータでも見え方が変わります。細かすぎるとばらばらに、粗すぎると形がつぶれて見えることがあります。

分布の偏りを読む

待ち時間のヒストグラムで右側に長い尾が伸びていれば、たまに非常に長く待つ人がいると読めます。平均だけでは見えにくい偏りを確認できます。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

横軸は階級、縦軸は度数または相対度数を表します。

棒グラフと違い、数値の範囲を連続した階級として扱うのが基本です。

階級幅、階級の境界、度数を確認して読みます。

分布の山、偏り、外れた値の有無を読み取る練習が重要です。

階級幅がそろっていない図では、単純に柱の高さだけで比較してよいかを確認します。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

相対度数

各階級の度数をデータ数 nnn で割ると、全体に占める割合として読めます。

relative frequency=class frequencyn\text{relative frequency}=\frac{\text{class frequency}}{n}relative frequency=nclass frequency​

記号の見方

  • nnn はデータ数です。
  • class frequency は、その階級に入るデータ数です。

定義の式

階級幅の目安

最大値と最小値の幅を階級数 kkk で割ると、階級幅の目安になります。実際には読みやすい区切りに調整します。

h=xmax⁡−xmin⁡kh=\frac{x_{\max}-x_{\min}}{k}h=kxmax​−xmin​​

記号の見方

  • hhh は階級幅の目安です。
  • kkk は階級数です。
  • xmax⁡x_{\max}xmax​ は最大値、xmin⁡x_{\min}xmin​ は最小値です。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

棒グラフと同じものだと思うこと

階級幅を確認せずに柱の高さだけを比べること

ヒストグラムから平均や中央値が正確に分かると思い込むこと

階級の境界にある値をどちらへ入れるか確認しないこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

ヒストグラムは、個々の値を階級にまとめて分布の形を見るための図です。

階級幅が同じなら、柱の高さを比べることで度数の多い範囲が分かります。

階級の取り方によって見え方が変わるため、図から結論を読むときは階級幅と境界を確認します。

ヒストグラムは分布の形を見る図であり、個々の観測値の順番や全ての正確な値を保存する図ではありません。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

箱ひげ図四分位数を使って、データの中心と広がりを短く表す図です。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

ヒストグラムは、データの形を眺めるための入口です。分布の山、偏り、広がりを図で見られるので、平均や標準偏差だけでは分かりにくい情報を補えます。

box-plot は分布を短く要約する図です。ヒストグラムで形を見て、箱ひげ図で代表的な位置と広がりを確認すると、データの読み取りが安定します。
