まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

第一種の過誤は、実際には異常がないのに警報を鳴らしてしまう状態です。

帰無仮説を棄却したからといって、いつも正しく差を見つけたとは限りません。

検定は確率的な手続きなので、基準どおりでもまれに強い偶然が起こります。

有意水準は、この誤警報の出しすぎを抑える役目です。

第一種の過誤を減らそうとすると、今度は本当の差を見逃しやすくなることがあります。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 帰無仮説が真のときの分布に棄却域を描くと、その面積が第一種の過誤に関係します。
  • 観測値がその棄却域に偶然入ってしまうと、第一種の過誤が起きます。
  • 有意水準を小さくすると、棄却域の面積が小さくなり、第一種の過誤は起こりにくくなります。
  • ただし、棄却域を狭くしすぎると、真の差があるときにも届きにくくなります。

定義

直感のあとで、言葉としての定義をここで固定します。

第一種の過誤とは、帰無仮説 H0H_0H0​ が実際には正しいのに、検定の結果として H0H_0H0​ を棄却してしまう誤りです。いわば「差がないのに差があると判断する」誤りで、偽陽性や false positive にあたります。有意水準 α\alphaα は、この誤りが起こる確率を大きくしすぎないように設定する基準です。第一種の過誤は、帰無仮説を棄却したときに必ず起こるわけではなく、H0H_0H0​ が真である場合に限って問題になります。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

仮説検定で有意水準の意味を説明する場面

「差があると判定したが、実は差がなかった」という誤りを問う場面

第一種の過誤と第二種の過誤を比較する場面

有意水準を厳しくしたときの影響を考える場面

効果がない薬を効くと判断する

実際には薬に効果がないのに、標本の偶然なぶれで p値が小さくなり、帰無仮説を棄却して「効果がある」と結論してしまうのが第一種の過誤です。

工場で誤警報を出す

製品の平均重量は基準どおりなのに、たまたま抜き取った標本が極端で、工程異常と判定してしまうのも第一種の過誤です。

有意水準 5% の意味

同じ手順の検定を帰無仮説が真の状況で何度も繰り返すと、理想化された条件ではそのうち約 5% 程度で第一種の過誤が起こりうるように、有意水準 0.05 を設定します。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

第一種の過誤は「帰無仮説が真なのに棄却する」誤りです。

有意水準 α\alphaα は、この誤りの確率を大きくしすぎないように決めます。

第一種の過誤は、棄却したときにだけ関係するわけではなく、帰無仮説が真かどうかで定義されます。

第一種の過誤と第二種の過誤は逆の誤りです。

第一種の過誤を抑えると、第二種の過誤が増えやすいことがあります。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

第一種の過誤確率

帰無仮説 H0H_0H0​ が真である状況で、検定統計量 TTT が棄却域 RRR に入ってしまう確率が第一種の過誤確率です。

P(T∈R∣H0)P(T\in R\mid H_0)P(T∈R∣H0​)

記号の見方

  • RRR は棄却域です。
  • TTT は検定統計量です。
  • ここでは、条件 H0H_0H0​ は「帰無仮説が真」という意味で読んでいます。

定義の式

有意水準との関係

検定は、第一種の過誤確率が有意水準 α\alphaα を超えないように設計されます。

P(T∈R∣H0)≤αP(T\in R\mid H_0)\le \alphaP(T∈R∣H0​)≤α

記号の見方

  • 連続分布の基本検定では、等号になることも多いです。
  • 離散的な検定では「以下」にしかならないことがあります。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

第一種の過誤を「対立仮説が真なのに棄却できない誤り」と取り違えること

有意水準そのものを、帰無仮説が真である確率だと考えること

p値が 0.05 未満なら第一種の過誤が絶対に起きないと思うこと

第一種の過誤を減らしても、他の誤りに影響しないと思うこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

第一種の過誤は、帰無仮説が真である世界で考える誤りです。したがって、対立仮説が真のときの見逃しとは別に扱います。

有意水準は、この第一種の過誤確率をコントロールするために導入されます。検定を繰り返し使ったときに、誤って棄却する割合が大きくなりすぎないようにする考え方です。

ただし、第一種の過誤を抑えるために棄却域を狭くすると、真の差がある場合にも棄却できず、第二種の過誤が増えやすくなります。

したがって、第一種の過誤は単独で考えるのではなく、第二種の過誤や検出力とのバランスで読む必要があります。

証明の流れを見る 第一種の過誤は、帰無仮説が真のときに観測統計量が棄却域へ入ることで起こります。
  1. まず、帰無仮説 H0H_0H0​ が真である状況を考えます。
  2. そのときの検定統計量 TTT の分布に対して、棄却域 RRR を決めます。
  3. もし観測値が偶然 RRR に入ると、H0H_0H0​ を棄却してしまいます。
  4. この確率 P(T∈R∣H0)P(T\in R\mid H_0)P(T∈R∣H0​) が、第一種の過誤確率です。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

第二種の過誤実際には差や効果があるのに、帰無仮説を棄却できず見逃してしまう誤りです。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

type-ii-error と対で覚えると、第一種の過誤は「ない差をあると判定する誤り」、第二種の過誤は「ある差を見逃す誤り」だと整理できます。試験では、この向きを逆にしないことが重要です。

また、この第一種の過誤をどこまで許すかを事前に決めるのが有意水準です。p値は観測後の結果ですが、第一種の過誤は検定手続きの設計と深く結びついています。
