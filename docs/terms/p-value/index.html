この記事でわかることを短く確認します。

  • 帰無仮説が正しいとしたときに、観測結果と同じかそれ以上に極端な結果が出る確率です。
  • 仮説検定で帰無仮説を棄却するかどうかを判断する場面

まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

p値は、「差がないとしたら、こんな結果はどれくらい珍しいか」という問いへの答えです。

値が小さいほど、帰無仮説では説明しにくい結果だと考えます。

値が大きいときは、帰無仮説でもそのくらいの結果は十分起こりうると読みます。

p値は観測データから計算するので、標本ごとに変わります。

片側検定か両側検定かで、「極端」の数え方が変わります。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 帰無仮説のもとでの分布に観測値を置き、その外側の尾の面積を p値と考えると理解しやすいです。
  • 右片側検定なら、観測値より右側に残る面積だけを見ます。
  • 両側検定なら、観測値と同じくらい遠い左右両方の面積を合わせて見ます。
  • 観測値が中央に近いほど尾の面積は大きくなり、端に寄るほど小さくなります。

定義

直感のあとで、言葉としての定義をここで固定します。

p値とは、帰無仮説 H0H_0H0​ が正しいと仮定したときに、観測された検定統計量と同じかそれ以上に極端な値が出る確率です。観測されたデータが帰無仮説とどれくらい両立しにくいかを数で表したもので、値が小さいほど「H0H_0H0​ のもとでは珍しい結果」と読みます。p値はデータを見たあとに計算する量であり、事前に決める有意水準 α\alphaα とは別物です。また、p値は「帰無仮説が真である確率」ではありません。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

仮説検定で帰無仮説を棄却するかどうかを判断する場面

検定統計量を計算したあとに、その珍しさを確率で読む場面

片側検定と両側検定で結果の読み方が変わることを確認する場面

有意水準との違いを問う統計検定2級の典型問題

p値 0.03 と有意水準 0.05

ある検定で p値が 0.03 だったとします。これは「帰無仮説のもとで、今の結果以上に極端な結果が出る確率が 3%」という意味です。もし有意水準が 0.05 なら棄却しますが、0.01 なら棄却しません。

コインが公平かを調べる

公平なコインを 20 回投げて 17 回表が出たとき、公平という帰無仮説のもとで 17 回以上表が出る確率を考えると、右片側の p値のイメージをつかみやすくなります。

片側と両側で変わる

同じ検定統計量 z=2.0z=2.0z=2.0 でも、右片側検定と両側検定では p値が違います。極端な値を右側だけ数えるか、左右両方数えるかが違うからです。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

p値は、帰無仮説が正しいとしたときの tail probability です。

p値はデータを見たあとに計算する量で、有意水準 α\alphaα は事前に決める量です。

p値が小さいほど、帰無仮説と観測データは両立しにくいと読みます。

p値は、帰無仮説が真である確率ではありません。

片側検定か両側検定かで p値の定義が変わります。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

右片側検定の p値

右片側検定では、観測された検定統計量 tobst_{\mathrm{obs}}tobs​ 以上に大きい値が出る確率を p値とします。

p=PH0(T≥tobs)p=P_{H_0}(T\ge t_{\mathrm{obs}})p=PH0​​(T≥tobs​)

記号の見方

  • TTT は帰無仮説のもとでの検定統計量です。
  • tobst_{\mathrm{obs}}tobs​ は実際の標本から得た観測値です。
  • 左片側検定では不等号の向きが逆になります。

定義の式

両側検定の p値

両側検定では、観測値と同じくらい極端な値を正負の両側で数えた確率を p値とします。

p=PH0(∣T∣≥∣tobs∣)p=P_{H_0}(|T|\ge |t_{\mathrm{obs}}|)p=PH0​​(∣T∣≥∣tobs​∣)

記号の見方

  • 対称な分布をもつ代表的な検定でよく使う書き方です。
  • 「極端」の範囲は、対立仮説の形に合わせて決まります。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

p値を「帰無仮説が真である確率」と読むこと

p値と有意水準を同じものだと考えること

p値が小さいほど効果量も大きいと決めつけること

片側検定と両側検定で同じ p値が出ると思うこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

p値は、検定統計量の観測値を帰無仮説の分布に当てはめて計算する tail probability です。したがって、p値の意味は必ず「帰無仮説のもとで」という条件つきです。

p値は観測データから計算されるため、同じ有意水準でも標本が変われば p値は変わります。これに対して、有意水準は検定前に固定する設計値です。

p値が小さいことは、統計的に珍しいことを意味しますが、効果が大きいことや実務上重要であることまで自動では意味しません。

片側検定と両側検定で p値が変わるのは、「どちらの方向を極端とみなすか」が対立仮説によって違うからです。

証明の流れを見る p値は、観測された検定統計量よりも極端な領域の確率を、帰無仮説の分布上で合計したものです。
  1. まず、観測データから検定統計量の値 tobst_{\mathrm{obs}}tobs​ を計算します。
  2. 次に、帰無仮説 H0H_0H0​ が正しいとしたときの検定統計量 TTT の分布を考えます。
  3. 対立仮説の向きに応じて、tobst_{\mathrm{obs}}tobs​ 以上または ∣tobs∣|t_{\mathrm{obs}}|∣tobs​∣ 以上に極端な領域を決めます。
  4. その領域の確率を足したものが p値です。
  5. したがって、p値は観測値そのものではなく、観測値の珍しさを確率へ変換した量だと分かります。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

帰無仮説仮説検定で最初の基準として置く仮説で、差がない・効果がない・基準どおりであることを表すことが多いです。有意水準帰無仮説が正しいのに棄却してしまう危険を、検定の前にどこまで許すかを表す基準値です。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

significance-level と一緒に読むと、p値は「データからあとで出てくる値」、有意水準は「先に決めておく基準」だと整理できます。この 2 つを入れ替えて読むのが、仮説検定で最も多い混同の 1 つです。

また、p値は必ず null-hypothesis を前提に計算されます。帰無仮説が何かを曖昧にしたまま p値だけ覚えると、意味を取り違えやすくなります。
