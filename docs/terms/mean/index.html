まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

平均は、ばらばらの値をならして1つの代表値にしたものです。

シーソーがつり合う位置を思い浮かべると、中心の感覚をつかみやすくなります。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 点数が 40 点、50 点、60 点なら、真ん中あたりの 50 点に全体の中心があると考えられます。
  • 値の重さを並べたときに、平均は左右がつり合う位置としてイメージできます。

定義

直感のあとで、言葉としての定義をここで固定します。

複数の値を合計して個数で割った値であり、データ全体を均したときの1つ分を表します。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

クラスの平均点を見て、全体の学力水準をざっくり把握するとき

月ごとの平均気温や平均売上のように、全体傾向を1つの数字にまとめたいとき

クラスの平均点

テスト結果をひとまとめにして、クラス全体がどのくらい取れているかを見るときに使います。

平均売上

日ごとの売上をならして、通常時の水準を把握するときに使います。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

平均は代表値だが、外れ値の影響を受けやすいです。

平均だけではデータのばらつきは分かりません。

分散や標準偏差と組み合わせて見ることが多いです。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

標本平均

\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i

観測値の合計をデータ数で割ると標本平均になります。

  • x_i は i 番目の観測値です。
  • n はデータ数です。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

平均だけでデータ全体の特徴を説明できたと思い込むこと

中央値と平均を同じものとして扱うこと

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

平均からの偏差をすべて足すと 0 になるため、平均はデータ全体のつり合いの点として理解できます。

証明の流れを見る 平均の定義を偏差の和に代入すると、偏差の総和が 0 になることを確認できます。
  1. \sum_{i=1}^{n}(x_i-\bar{x}) を書き下します。
  2. \bar{x} = \frac{1}{n}\sum_{i=1}^{n}x_i を代入します。
  3. 合計を整理すると 0 になります。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

分散データが平均のまわりにどのくらい散らばっているかを、二乗を使って測る代表的な指標です。標準偏差分散を元の単位に戻して解釈しやすくした、散らばりの代表的な指標です。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

平均は、値を1つに要約する入口の概念です。平均を見たあとは、その周りにどれくらい散らばっているかを分散や標準偏差で見る流れが自然です。
