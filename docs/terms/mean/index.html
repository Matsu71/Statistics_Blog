まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

平均は、ばらばらの値を同じ大きさに均して、全体を 1 つの代表値にしたものです。

シーソーがつり合う位置を思い浮かべると、平均が「左右のずれが打ち消し合う点」だとつかみやすくなります。

平均は値の順番ではなく、合計と個数で決まります。並べ方を変えても、合計と個数が同じなら平均は変わりません。

平均は中心の目安ですが、データの中に実際に存在する値とは限りません。たとえば 2, 3, 7 の平均は 4 です。

標本平均と母平均は名前が似ていますが、手元の標本の平均なのか、知りたい母集団全体の平均なのかを見分けることが大切です。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 40, 40, 70 の合計は 150 です。3 人に同じように配り直すと 50, 50, 50 になり、この 50 が平均です。
  • 値を数直線上に置いたとき、平均より小さい側の不足分と、平均より大きい側の余りが打ち消し合います。
  • 40, 50, 60 も 40, 40, 70 も平均は 50 です。平均は「値の並び方」ではなく、「合計を均した結果」を見ます。
  • 同じ平均でも、値が平均の近くに集まっているか、大きく散らばっているかは別の情報です。

定義

直感のあとで、言葉としての定義をここで固定します。

平均は、複数の値を合計して個数で割った値であり、データ全体を同じ大きさに均したときの 1 つ分を表します。全体の中心を 1 つの数でつかむには便利ですが、散らばりの大きさ、値の並び方、最も多い値までは表しません。標本でも母集団でも同じ形で考えられますが、どの集団の平均なのかを必ず確認します。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

クラスの平均点を見て、全体の学力水準をざっくり把握するとき

月ごとの平均気温や平均売上のように、全体傾向を 1 つの数字にまとめたいとき

標本平均や母平均を比べて、推定や検定の入口に入るとき

平均は同じでも、ばらつきや外れ値の有無を追加で確認したいとき

クラスの平均点

テスト結果をひとまとめにして、クラス全体がどのくらい取れているかを見るときに使います。平均点を見ると、全体の中心が一目で分かります。

例題: 3つの値の平均

60 点、70 点、80 点の平均は、合計 210 点を 3 で割って 70 点です。まず「合計して、個数で割る」という計算の型を確認します。

外れ値が入ると平均は動く

50, 50, 50 の平均は 50 ですが、50, 50, 100 になると平均は約 66.7 になります。1 つ大きい値が入るだけで、中心の目安が右へ引っ張られます。

中央値・最頻値との比較

50, 50, 100 では、平均は約 66.7、中央値は 50、最頻値も 50 です。平均は合計のつり合い、中央値は並べたときの真ん中、最頻値は最も多く現れる値を見ています。

同じ平均でも形は違う

10, 20, 30 と 10, 10, 40 はどちらも平均 20 ですが、値の散らばり方は違います。平均だけでは、データの形やばらつきまでは分かりません。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

平均は代表値だが、外れ値の影響を受けやすいです。1 つの値が大きく変わると、その変化は平均にも反映されます。

平均だけではデータのばらつき、偏り、最も多い値は分かりません。

分散や標準偏差と組み合わせて見ることが多いです。

問題文で「平均」とだけあれば、何の集団の平均かを確認します。

平均は合計のつり合い、中央値は順位、最頻値は出現回数を見ます。3 つの代表値の役割を区別します。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

算術平均

観測値の合計をデータ数で割ると平均になります。値をすべて集めてから同じ大きさに配り直した「1つ分」が平均です。

xˉ=1n∑i=1nxi\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_ixˉ=n1​i=1∑n​xi​

記号の見方

  • xix_ixi​ は i 番目の観測値です。
  • nnn はデータ数です。
  • 標本でも母集団でも、同じ形の式で平均を考えます。

定義の式

偏差の和

各値から平均を引いた差を偏差といいます。平均を基準にすると、平均より上の差と下の差が全体で打ち消し合います。

∑i=1n(xi−xˉ)=0\sum_{i=1}^{n}(x_i-\bar{x}) = 0i=1∑n​(xi​−xˉ)=0

記号の見方

  • xi−xˉx_i-\bar{x}xi​−xˉ は i 番目の値の偏差です。
  • 正の偏差と負の偏差が合わさって 0 になります。
  • この性質が、平均をつり合いの点として見る根拠になります。

定義の式

代表値を任意の値 aaa とすると、二乗誤差の合計は平均 xˉ\bar{x}xˉ を選んだときに最小になります。平均は二乗のずれを小さくする自然な中心でもあります。

  • したがって、二乗誤差の合計は a=xˉa=\bar{x}a=xˉ のとき最小になります。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

平均だけでデータ全体の特徴を説明できたと思い込むこと

中央値と平均を同じものとして扱うこと

標本平均と母平均の区別をしないこと

平均がデータの中に必ず存在する値だと思い込むこと

外れ値があっても平均はあまり動かないと考えてしまうこと

平均が同じなら、データの散らばり方も同じだと考えること

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

平均からの偏差をすべて足すと 0 になるため、平均はデータ全体のつり合いの点として理解できます。

平均は「足した値を個数で均したもの」なので、合計を分かりやすく 1 つの代表値に変える役割があります。

平均は全ての値を使うので、極端に大きい値や小さい値があると動きやすくなります。一方で、中央値は値を順番に並べたときの位置を見るため、外れ値の影響が平均より小さいことがあります。

平均が同じでも、データの散らばり方は同じとは限りません。散らばりは分散や標準偏差で別に確認します。

補足として、平均は二乗誤差の合計を最小にする値でもあります。これは、平均が「合計を均した点」であるだけでなく、「二乗のずれを最も小さくする点」でもあることを意味します。

標本平均は母平均の推定に使われますが、1 回の標本だけで母集団の中心が完全に分かるわけではありません。

証明の流れを見る 平均の定義を偏差の和に代入すると、偏差の総和が 0 になることを段階的に確認できます。

  2. 偏差の和を分けると、∑i=1n(xi−xˉ)=∑i=1nxi−∑i=1nxˉ\sum_{i=1}^{n}(x_i-\bar{x}) = \sum_{i=1}^{n}x_i - \sum_{i=1}^{n}\bar{x}∑i=1n​(xi​−xˉ)=∑i=1n​xi​−∑i=1n​xˉ です。
  3. xˉ\bar{x}xˉ はすべての iii で同じ値なので、∑i=1nxˉ=nxˉ\sum_{i=1}^{n}\bar{x} = n\bar{x}∑i=1n​xˉ=nxˉ です。
  4. 平均の定義から nxˉ=∑i=1nxin\bar{x} = \sum_{i=1}^{n}x_inxˉ=∑i=1n​xi​ なので、偏差の和は ∑i=1nxi−nxˉ=0\sum_{i=1}^{n}x_i - n\bar{x} = 0∑i=1n​xi​−nxˉ=0 になります。
  5. したがって、平均より大きいずれと小さいずれは全体で打ち消し合い、平均をつり合いの点として読めます。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

分散データが平均のまわりにどのくらい散らばっているかを、二乗を使って測る代表的な指標です。標準偏差分散を元の単位に戻して解釈しやすくした、散らばりの代表的な指標です。標本平均標本に含まれる観測値を合計して個数で割った、標本を代表する平均です。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

平均は、値を 1 つに要約する入口の概念です。sample-mean は標本に対して計算した平均で、mean の考え方が実際の調査データにどう使われるかを示す例です。

平均を見たあとは、その周りにどれくらい散らばっているかを variance や standard-deviation で見る流れが自然です。平均は中心、分散と標準偏差は散らばりを見るための指標だと分けると、問題文の意味が追いやすくなります。

中央値や最頻値は、現時点では未公開ページですが、平均との比較で重要です。中央値は順番で見た真ん中、最頻値は最も多く出る値、平均は合計を均した値として区別しておくと、外れ値がある問題で混乱しにくくなります。
