大数の法則

独立に同じ分布から標本を増やしていくと、標本平均が母平均に近づいていくという法則です。

30秒で要点

この記事でわかることを短く確認します。

  • 独立に同じ分布から標本を増やしていくと、標本平均が母平均に近づいていくという法則です。
  • 標本平均を母平均の推定に使う意味を説明する場面
  • 大数の法則は、標本平均が母平均に近づく法則です。個々の観測値についての法則ではありません。

まずはイメージで理解する

式の前に、この用語が何を見ているのかを言葉でつかみます。

コイン投げを続けると、表の割合は最初は大きく揺れますが、回数が増えるほど 0.5 の近くに落ち着きやすくなります。

新しい観測値が 1 つ増えても、平均に与える影響は標本サイズが大きいほど小さくなります。

大数の法則が言っているのは、個々の値が安定することではなく、平均が安定することです。

近づくとは「ずれが小さい確率が高くなる」という意味で、毎回まっすぐ真値へ向かうことは保証しません。

平均を推定に使う理由のひとつは、この法則で標本平均の安定性が支えられているからです。

視覚的な説明

画像がなくても読めるように、簡易図と短い説明をセットにしています。

  • 累積平均のグラフを描くと、最初は大きく上下しても、試行回数が増えるほど線の動きがゆるやかになります。
  • 10 回の平均より 100 回、1000 回の平均のほうが、母平均の近くに集まりやすいです。
  • コイン投げの表の割合を横軸に試行回数、縦軸に累積割合で描くと、0.5 の近くにだんだん張り付くイメージがつかめます。
  • 図では、個々の観測値の折れ線ではなく、累積平均の折れ線を見ることが重要です。

定義

直感のあとで、言葉としての定義をここで固定します。

大数の法則とは、平均 μ\muμ をもつ確率変数から独立に標本を取り続けると、標本平均 Xˉn\bar{X}_nXˉn​ が母平均 μ\muμ に近づいていくという法則です。ここで大切なのは、「標本数が増えるほど、標本平均が母平均から大きく外れる確率が小さくなる」という意味であり、毎回単調に近づくとか、いつか必ず完全一致するという意味ではありません。1 回ごとの観測値ではなく、平均の安定性について述べる法則です。

現実でどう現れるか

試験の文脈と日常・実務の文脈を行き来しながら、この用語の居場所を確認します。

よく出てくる場面

標本平均を母平均の推定に使う意味を説明する場面

大きい標本ほど平均が安定しやすいことを説明する場面

中心極限定理との違いを問う場面

標本数を増やすと推定がどう変わるかを直感的に理解する場面

コイン投げの割合

公平なコインなら、10 回投げたときの表の割合は 0.3 や 0.7 になることがありますが、1000 回投げると 0.5 の近くに集まりやすくなります。これは 0 と 1 の標本平均が 0.5 に近づく例です。

サイコロの平均

サイコロを 5 回だけ振った平均は 2.2 や 4.8 にぶれることがありますが、500 回の平均は理論上の母平均 3.5 にかなり近づきやすくなります。

アンケート平均点

満足度調査で回答者数が少ないと平均点は大きく動きやすいですが、無作為に集めた回答が増えると、平均点は母集団全体の平均に安定して近づきやすくなります。

試験で重要なポイント

時間がないときは、この節だけでも先に確認すると着眼点を押さえやすくなります。

大数の法則は、標本平均が母平均に近づく法則です。個々の観測値についての法則ではありません。

「近づく」は、標本数が増えるほど外れにくくなるという意味であり、毎回真値へ単調に近づく意味ではありません。

標準的な学習では、独立で同じ分布に従い、少なくとも平均が存在する標本を前提にします。

中心極限定理は分布の形を述べますが、大数の法則は近づく先が母平均であることを述べます。

標本数を増やすことは平均の安定性を高めますが、偏った標本抽出の問題まで自動で解決するわけではありません。

数式とその意味

式は単独で覚えるのではなく、記号の意味と一緒に確認します。

定義の式

標本平均の収束

標本サイズ nnn を増やしていくと、標本平均 Xˉn\bar{X}_nXˉn​ は確率的に母平均 μ\muμ へ近づいていくことを表します。

Xˉn→Pμ\bar{X}_n \xrightarrow{P} \muXˉn​P​μ

記号の見方

  • →P\xrightarrow{P}P​ は確率収束を表します。
  • Xˉn\bar{X}_nXˉn​ は大きさ nnn の標本平均です。
  • 統計検定2級では、独立で同じ分布に従う標本を基本形として考えます。

定義の式

ずれが大きい確率の表し方

任意の正の幅 ε\varepsilonε に対して、標本平均が母平均から ε\varepsilonε 以上ずれる確率は、標本サイズが大きくなると 0 に近づきます。

P(∣Xˉn−μ∣>ε)→0(n→∞)P(|\bar{X}_n-\mu|>\varepsilon)\to 0 \quad (n\to\infty)P(∣Xˉn​−μ∣>ε)→0(n→∞)

記号の見方

  • ε>0\varepsilon>0ε>0 は許したい誤差幅です。
  • 完全一致を言う式ではなく、外れにくくなることを表す式です。

よくある勘違い

似た概念との違いをここで整理しておくと、試験でも混線しにくくなります。

大数の法則を、各観測値が真値へ近づく法則だと思うこと

標本平均が毎回単調に母平均へ近づくと考えること

いつか必ず母平均と完全一致する、と読み替えること

中心極限定理と同じ内容だと思うこと

標本数が多ければ、偏った抽出でも必ず正しい平均になると考えること

厳密な説明・証明

主導線を邪魔しないように、詳しい説明は折りたたみで分けています。

厳密な説明を読む 定義の背景や前提を、もう一歩ていねいに確認します。

大数の法則は「平均がどこへ向かうか」を述べる法則です。標本平均の中心が母平均 μ\muμ にあるだけでなく、標本サイズが増えるとその周りのぶれが小さくなることを意味します。

確率収束 Xˉn→Pμ\bar{X}_n \xrightarrow{P} \muXˉn​P​μ とは、どんな小さい誤差幅 ε\varepsilonε を決めても、その幅から外れる確率を十分小さくできるという意味です。

有限分散 σ2\sigma^2σ2 があると、Var(Xˉn)=σ2/n\mathrm{Var}(\bar{X}_n)=\sigma^2/nVar(Xˉn​)=σ2/n となり、標本平均のぶれが nnn とともに小さくなることが見えます。

大数の法則は、標本平均が母平均へ近づくことを支えますが、近づいたまわりの形が正規分布に近いかどうかまでは述べません。そこは中心極限定理の役割です。

証明の流れを見る 有限分散を仮定すると、チェビシェフの不等式から標本平均が母平均へ確率収束することが確かめられます。
  1. X1,…,XnX_1,\dots,X_nX1​,…,Xn​ が独立で同じ分布に従い、平均 μ\muμ、分散 σ2<∞\sigma^2<\inftyσ2<∞ をもつとします。
  2. 標本平均 Xˉn=1n∑i=1nXi\bar{X}_n=\frac{1}{n}\sum_{i=1}^{n}X_iXˉn​=n1​∑i=1n​Xi​ の平均は E[Xˉn]=μE[\bar{X}_n]=\muE[Xˉn​]=μ です。
  3. 独立性より、Var(Xˉn)=σ2/n\mathrm{Var}(\bar{X}_n)=\sigma^2/nVar(Xˉn​)=σ2/n です。
  4. チェビシェフの不等式から、任意の ε>0\varepsilon>0ε>0 について P(∣Xˉn−μ∣≥ε)≤σ2nε2P(|\bar{X}_n-\mu|\ge \varepsilon)\le \frac{\sigma^2}{n\varepsilon^2}P(∣Xˉn​−μ∣≥ε)≤nε2σ2​ です。
  5. 右辺は n→∞n\to\inftyn→∞ で 0 に近づくので、P(∣Xˉn−μ∣≥ε)→0P(|\bar{X}_n-\mu|\ge \varepsilon)\to 0P(∣Xˉn​−μ∣≥ε)→0 となります。
  6. したがって、Xˉn→Pμ\bar{X}_n \xrightarrow{P} \muXˉn​P​μ、つまり標本平均は母平均へ確率収束します。

関連用語

1語だけで閉じず、近い概念へ横移動すると理解がつながりやすくなります。

中心極限定理独立で同じ分布に従う標本を十分多く平均すると、元の分布が正規分布でなくても標本平均はおおよそ正規分布に従うという定理です。

補足ノート

本文で伝えきれない背景や、周辺概念とのつながりを短く補います。

central-limit-theorem と並べて読むと違いが整理しやすくなります。大数の法則は「標本平均が母平均へ近づく」という位置の話で、中心極限定理は「その近づいたまわりでどんな形にぶれるか」という分布の話です。

標本平均を推定に使う感覚がまだ曖昧なときは、コイン投げやサイコロの平均の例で「回数が増えるほど平均が安定する」感覚を先に押さえると理解しやすくなります。
