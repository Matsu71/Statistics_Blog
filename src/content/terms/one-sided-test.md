short_definition: 対立仮説で差の向きを一方向にだけ指定し、その側の尾だけで判定する仮説検定です。
definition: 片側検定とは、対立仮説で「基準より大きい」または「基準より小さい」のどちらか一方向だけを問題にする仮説検定です。右片側検定なら大きい方向、左片側検定なら小さい方向だけを見ます。そのため、有意水準 $\alpha$ は片側の尾にまとめて置かれます。片側検定は、研究上その方向だけに意味があると事前に決まっているときに使います。観測後に都合よく片側検定へ切り替えてはいけません。
intuition:
  - 片側検定は、「大きいか」または「小さいか」のどちらかだけを聞く検定です。
  - 反対方向のずれは、たとえ大きくても、この検定では対立仮説を支持する証拠になりません。
  - 有意水準を片側だけに使うので、指定した方向には敏感になります。
  - ただし、指定と逆向きの差も重要なら、片側検定では不十分です。
  - 方向は標本を見る前に決める必要があります。
visual_explanation:
  - 右片側検定では、帰無仮説の分布の右端だけを棄却域にします。
  - 左片側検定では、左端だけを棄却域にします。
  - 観測値が反対側の端にあっても、その方向を問題にしていなければ棄却理由にはなりません。
  - 両側検定が両端に有意水準を分けるのに対し、片側検定は片側にまとめて置きます。
where_it_appears:
  - 新しい教材で平均点が上がったかだけを調べたい場面
  - 工程改善で不良率が下がったかだけを確認したい場面
  - 平均値が基準を超えているかだけを監視したい品質管理の場面
  - 片側と両側で臨界値や p値の意味がどう変わるかを問う試験問題
practical_examples:
  - title: 新教材で平均点が上がったか
    description: 旧来の平均点 70 点に対し、新教材で上がったかだけを見たいなら $H_1:\mu>70$ と置きます。標本から $Z=1.8$ が出たとき、$\alpha=0.05$ の右片側検定なら臨界値は約 1.645 なので棄却できます。
  - title: 不良率が下がったか
    description: 工程改善後に不良率が下がったかだけを確かめるなら、対立仮説は $H_1:p<p_0$ です。上がる方向の差は、この片側検定で「改善の証拠」にはなりません。
  - title: 逆方向のずれは支持にならない
    description: $H_1:\mu>70$ で検定しているのに標本平均が 66 点へ大きく下がっていた場合、その結果は右片側検定の対立仮説を支持しません。大きく外れていても、向きが違えば別問題です。
exam_points:
  - 片側検定は、対立仮説が一方向の不等号で書かれるときに使います。
  - 有意水準 $\alpha$ は片側の尾に全部置かれます。
  - 同じ $\alpha$ なら、指定方向では両側検定より棄却しやすくなります。
  - 反対方向のずれは、片側検定では対立仮説の根拠になりません。
  - 片側か両側かはデータを見る前に決めます。
formulas:
  - name: 右片側検定
    latex: 'H_1:\mu>\mu_0,\quad R=\{z \mid z>z_{1-\alpha}\}'
    description: 母平均が基準値より大きいかを調べる右片側検定では、上側の尾だけを棄却域 $R$ にします。
    conditions:
      - '$R$ は棄却域です。'
      - '$z_{1-\alpha}$ は標準正規分布の累積確率 $1-\alpha$ に対応する臨界値です。'
      - z 検定での代表的な形です。
  - name: 左片側検定
    latex: 'H_1:\mu<\mu_0,\quad R=\{z \mid z<-z_{1-\alpha}\}'
    description: 母平均が基準値より小さいかを調べる左片側検定では、下側の尾だけを棄却域にします。
    conditions:
      - 上側の大きな値は、この検定では棄却理由になりません。
      - 右片側と左右が入れ替わった形です。
rigorous_explanation:
  - 片側検定では、対立仮説が指定した方向にだけ有意水準を使います。そのため、同じ $\alpha$ と同じ検定統計量を使う両側検定より、指定方向の臨界値は中心に近くなります。
  - たとえば標準正規分布なら、$\alpha=0.05$ の右片側検定の臨界値は約 1.645 ですが、両側検定では約 1.96 です。この違いが、指定方向では片側検定のほうが棄却しやすい理由です。
  - ただし、この性質は「最初からその方向だけに意味がある」と決めていることが前提です。観測後に片側へ変更すると、有意水準の解釈が崩れます。
  - 反対方向の差が重要な研究では、片側検定ではなく両側検定を選ぶべきです。
proof:
  summary: 同じ有意水準なら、片側検定はその有意水準を片側だけに置くので、指定方向では両側検定より臨界値が小さくなります。
  outline_steps:
    - '標準正規分布を使う検定を考え、全体の有意水準を $\alpha=0.05$ とします。'
    - '右片側検定では、上側 5% 全部を棄却域にするので臨界値は $z_{0.95}\approx1.645$ です。'
    - '両側検定では、上側 2.5% と下側 2.5% に分けるので臨界値は $z_{0.975}\approx1.96$ です。'
    - 'したがって、右方向だけを見るなら、片側検定のほうが小さい統計量でも棄却に届きます。'
common_mistakes:
  - データを見たあとで、都合のよい方向に片側検定を選ぶこと
  - 片側検定なら反対方向の大きなずれも有意だと思うこと
  - 「差があるか」という問いに片側検定を当てはめること
  - 片側検定の p値と両側検定の p値を同じ意味で比べること

`two-sided-test` と比べると、片側検定は「どちら向きの差に意味があるか」をはっきり限定した検定です。反対方向も重要なら、片側検定ではなく両側検定で読む必要があります。

この違いは `alternative-hypothesis` の書き方から決まります。問題文が「上がったか」「下がったか」なのか、「違うか」なのかを先に確認すると、向きの選び間違いを防ぎやすくなります。
