short_definition: 独立な標準正規変数の二乗和でできる、0 以上の値だけをとる確率分布です。
definition: カイ二乗分布は、独立な標準正規変数 $Z_1,\dots,Z_k$ の二乗和 $Z_1^2+\cdots+Z_k^2$ で定義される確率分布で、自由度 $k$ を持ちます。二乗和なので値は必ず 0 以上になり、自由度が小さいと右に長い裾を持つ形になります。正規母集団の分散に関する推定や検定、分割表の検定統計量などで重要な役割を持ちます。
intuition:
  - 標準化したずれを二乗して足し合わせた量だと考えると、カイ二乗分布の意味がつかみやすいです。
  - 二乗しているので負の値は出ません。
  - 自由度が大きくなるほど、二乗して足す項が増え、分布の中心は右へ動きます。
  - 各項が「平均との差の大きさ」を表しているので、全体としてずれが大きいほどカイ二乗値も大きくなります。
  - カイ二乗分布は正規分布とは違い、左右対称ではありません。
visual_explanation:
  - 横軸の 0 から始まり、右に裾を引く分布として描かれます。
  - 自由度 1 や 2 では強く右に歪みますが、自由度が増えると山がなだらかになっていきます。
  - いくつもの二乗したずれを足し合わせた面積や大きさを見ている、と考えるとイメージしやすいです。
where_it_appears:
  - 正規母集団の分散に関する区間推定や検定
  - 適合度検定や独立性の検定などのカイ二乗検定
  - F 分布を作る材料として
practical_examples:
  - title: 標準正規変数の二乗和
    description: 独立な標準正規変数 $Z_1,Z_2$ があり、たまたま $Z_1=1.2$、$Z_2=-0.5$ なら、$Z_1^2+Z_2^2=1.44+0.25=1.69$ です。カイ二乗分布は、このような二乗和が確率的にどう動くかを表します。
  - title: 分散の検定統計量
    description: 正規母集団からの標本では、$\frac{(n-1)S^2}{\sigma^2}$ がカイ二乗分布に従います。標本分散 $S^2$ が母分散 $\sigma^2$ と比べてどれくらい大きいかを、自由度つきの分布で判断します。
  - title: 分割表でのずれ
    description: 分割表の各セルで、観測度数と期待度数の差を二乗して期待度数で割り、全体で足し合わせるとカイ二乗統計量になります。期待からのずれが全体としてどれだけ大きいかを見る考え方です。
exam_points:
  - カイ二乗分布は、独立な標準正規変数の二乗和で定義されます。
  - 値は 0 以上で、自由度 $k$ によって形が変わります。
  - 平均は $k$、分散は $2k$ です。
  - 正規母集団の分散の問題では、$\frac{(n-1)S^2}{\sigma^2}$ がカイ二乗分布に従います。
  - F 分布は 2 つの独立なカイ二乗分布から作られます。
formulas:
  - name: カイ二乗分布の定義
    latex: 'X=\sum_{i=1}^{k}Z_i^2 \sim \chi^2(k)'
    description: 独立な標準正規変数の二乗和が、自由度 $k$ のカイ二乗分布です。
    conditions:
      - '$Z_1,\dots,Z_k$ は独立で、それぞれ $N(0,1)$ に従います。'
      - '$k$ が自由度です。'
  - name: 平均と分散
    latex: 'E[X]=k,\quad \operatorname{Var}(X)=2k'
    description: 自由度 $k$ のカイ二乗分布の中心と広がりを表す基本性質です。
    conditions:
      - '$X\sim\chi^2(k)$ のときに成り立ちます。'
  - name: 標本分散との関係
    latex: '\frac{(n-1)S^2}{\sigma^2}\sim \chi^2(n-1)'
    description: 正規母集団からの標本では、標本分散を母分散で標準化した統計量がカイ二乗分布に従います。分散の推定や検定の土台になる式です。
    conditions:
      - 正規母集団からの標本であることを前提にします。
      - '$S^2$ は不偏分散、$\sigma^2$ は母分散です。ここでは手元のデータの分散とは区別します。'
rigorous_explanation:
  - カイ二乗分布は、標準正規変数の二乗和で定義されるので、値は必ず 0 以上です。この性質は分散のような「大きさ」を扱う量と相性がよいです。
  - 自由度は、独立な標準正規変数をいくつ足したかを表します。自由度が増えると、分布の中心は大きくなります。
  - 正規母集団の標本分散にカイ二乗分布が現れるため、分散に関する区間推定や検定ではこの分布が重要になります。
  - F 分布は、独立な 2 つのカイ二乗分布を自由度で割って比にしたものです。したがって、カイ二乗分布を理解すると F 分布にもつながります。
proof:
  summary: カイ二乗分布の基本的な形と平均・分散は、独立な標準正規変数の二乗和という定義から読み取れます。
  outline_steps:
    - '$X=\sum_{i=1}^{k}Z_i^2$ とおくと、各 $Z_i^2$ は 0 以上なので、$X$ も 0 以上です。'
    - '$Z_i$ は独立な標準正規変数なので、$E[Z_i^2]=1$ です。'
    - '期待値の線形性から、$E[X]=\sum_{i=1}^{k}E[Z_i^2]=k$ となります。'
    - '$Z_i^2$ の分散は 2 で、独立性より分散も足せるので、$\operatorname{Var}(X)=2k$ です。'
    - 'このように、二乗和という定義だけで、カイ二乗分布の基本性質が確認できます。'
common_mistakes:
  - カイ二乗分布を正規分布のように左右対称だと考えること
  - カイ二乗値が負になることもあると思うこと
  - 自由度を、いつも標本サイズそのものだと勘違いすること
  - 標本分散との関係式を、正規母集団という前提なしに使うこと

`f-distribution` は、独立な 2 つのカイ二乗分布を自由度で割って比にした分布です。したがって、カイ二乗分布で「二乗したずれを足し合わせる」「自由度で形が変わる」という感覚をつかんでおくと、F 分布の意味も追いやすくなります。

カイ二乗分布は、平均のまわりのずれを二乗して集める場面でよく現れます。分散や検定統計量に出てくるのは、その量が 0 以上で、ずれの大きさをまとめているからです。
