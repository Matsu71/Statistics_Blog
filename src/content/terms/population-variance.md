short_definition: 母集団全体が母平均のまわりにどのくらい散らばっているかを表す、母集団のばらつきの指標です。
definition: 母分散は、母集団に含まれる全ての値が母平均 $\mu$ からどのくらい離れているかを二乗で測り、その平均をとった値です。記号ではふつう $\sigma^2$ で表します。母集団が定まれば母分散も 1 つに決まる固定された母数であり、標本から計算する標本分散や不偏分散とは区別して考えます。単位は元データの二乗になるため、解釈では標準偏差とあわせて見ることも多いです。
intuition:
  - 母分散は、母集団全体が母平均の近くにまとまっているか、広く散っているかを見る量です。
  - 母平均が同じでも、値が母平均の近くに集まる母集団と遠くまで広がる母集団では母分散が違います。
  - 偏差をそのまま足すと打ち消し合うので、二乗してから平均することで散らばりの大きさを表します。
  - 母集団全体を使って計算するので、標本を取り直すたびに変わる量ではありません。
  - すべての値が同じなら、母平均からのずれは全部 0 なので母分散も 0 です。
visual_explanation:
  - 母平均のまわりに点がぎゅっと集まっている図では、母分散は小さくなります。
  - 同じ母平均でも、左右に大きく広がった図では母分散が大きくなります。
  - 2, 4, 6, 8 の母平均は 5 で、母平均からの距離は 3, 1, 1, 3 です。遠い点があるほど、二乗によって母分散へ強く効きます。
  - 点が全部同じ位置に重なっていれば、母平均からの距離がないので母分散は 0 になります。
where_it_appears:
  - 母集団全体のばらつきを母数として定義するとき
  - 標本分散や不偏分散が、何を近似しようとしているかを整理するとき
  - 正規分布などで、中心と散らばりの母数を区別するとき
  - 推定や検定で、母分散が既知か未知かを条件として読むとき
practical_examples:
  - title: 4 つの値から母分散を求める
    description: 母集団が 2, 4, 6, 8 の 4 つ全体だとすると、母平均は 5、偏差平方は 9, 1, 1, 9 です。これらの平均は $(9+1+1+9)/4=5$ なので、母分散は 5 です。
  - title: '確認: 母平均が同じでも散らばりは違う'
    description: 母集団 A が 4, 5, 6、母集団 B が 1, 5, 9 なら、どちらも母平均は 5 です。しかし B のほうが母平均から遠い値を含むので、母分散は大きくなります。
  - title: 全て同じ値の母集団
    description: 母集団が 7, 7, 7, 7 なら、母平均は 7 で全ての偏差は 0 です。このとき母分散も 0 になり、散らばりがないことを示します。
exam_points:
  - 母分散は母集団全体に対して定まる母数で、記号はふつう $\sigma^2$ です。
  - 式では母平均 $\mu$ を使い、母集団の大きさ $N$ で割ります。
  - 標本分散や不偏分散は標本から計算する量であり、母分散そのものではありません。
  - 母分散の単位は元データの単位ではなく、その二乗になります。
  - 母分散が 0 なのは、母集団の全ての値が同じときだけです。
  - 平均の違いと散らばりの違いを分けて読むことが試験では重要です。
formulas:
  - name: 母分散
    latex: '\sigma^2 = \frac{1}{N}\sum_{i=1}^{N}(x_i-\mu)^2'
    description: 母集団全体の各値 $x_i$ が母平均 $\mu$ からどれだけ離れているかを二乗し、その平均をとると母分散 $\sigma^2$ になります。母集団全体の広がりを 1 つの数にまとめた式です。
    conditions:
      - '$N$ は母集団の大きさです。'
      - '$\mu$ は母平均です。'
      - 母集団全体の全ての値を使って計算します。
  - name: 展開形
    latex: '\sigma^2 = \frac{1}{N}\sum_{i=1}^{N}x_i^2-\mu^2'
    description: 母分散は、二乗した値の平均から母平均の二乗を引いた形にも整理できます。母分散が「二乗の大きさ」と「中心」の差を見ていることが分かる式です。
    conditions:
      - '$\frac{1}{N}\sum_{i=1}^{N}x_i^2$ は母集団の二乗平均です。'
      - '$\mu^2$ は母平均の二乗です。'
      - 同じ母集団について成り立つ恒等式です。
  - name: 母標準偏差との関係
    latex: '\sigma = \sqrt{\sigma^2}'
    description: 母分散の平方根をとると母標準偏差になります。二乗の単位を元の単位へ戻して解釈しやすくした量です。
    conditions:
      - '$\sigma$ は母標準偏差です。'
      - 母分散と母標準偏差は、どちらも母集団全体の散らばりを表します。
rigorous_explanation:
  - 母分散は母集団に属する母数なので、母集団が固定されれば値も固定されます。標本を取り直して変動するのは、標本から作る統計量のほうです。
  - 偏差の和 $\sum_{i=1}^{N}(x_i-\mu)$ は 0 になるため、散らばりを測るには偏差をそのまま足すだけでは足りません。そこで二乗を使って、大きなずれほど強く反映させます。
  - 母分散は 0 以上で、0 になるのは全ての母集団の値が母平均と一致するときだけです。少なくとも 1 つでも異なる値があれば、どこかの偏差平方が正になります。
  - 実際には母集団全体が見えないことが多いため、標本から計算した `sample-variance` や `unbiased-variance` を通じて母分散を考えます。
  - 標本分散は手元の標本の散らばりを記述する量で、不偏分散は母分散を推定する文脈で補正した量です。どの量を使っているかを問題文で確認することが大切です。
common_mistakes:
  - 母分散の式で $N$ ではなく $n-1$ を使ってしまうこと
  - 母平均 $\mu$ ではなく標本平均 $\bar{x}$ をそのまま代入してしまうこと
  - 母分散と標本分散、不偏分散を区別せずに同じ言葉で扱ってしまうこと
  - 母分散の数値を、元データと同じ単位のまま読んでしまうこと
  - 母平均が大きいことと母分散が大きいことを混同してしまうこと

`sample-variance` は手元の標本から計算した散らばりで、`population-variance` は知りたい母集団全体の散らばりです。標本と母集団の区別が曖昧だと、どの式を使うべきかも曖昧になります。

母分散を推定したい場面では `unbiased-variance` が続きます。標本分散をそのまま読む場面なのか、母分散を推定するために補正する場面なのかを分けて考えると、試験問題を整理しやすくなります。
