title: 共分散
slug: covariance

level: standard

category: correlation_regression
short_definition: 2つの変量が一緒に大きくなりやすいか、逆向きに動きやすいかを、平均からのずれの積で表す量です。
definition: 共分散は、対応する 2 つのデータ $x_i$ と $y_i$ が、それぞれの平均 $\bar{x}$、$\bar{y}$ からどの向きにどれくらいずれるかをかけ合わせ、その平均をとった量です。両方が平均より上、または両方が平均より下になりやすければ正になり、片方が上で片方が下になりやすければ負になります。ただし値の大きさは単位に依存するため、強さをそのまま比較するより、向きと一緒に動く傾向の有無を読む入口として使います。
intuition:
  - 各観測で、$x$ も $y$ も平均より上なら、ずれの積は正になります。
  - $x$ が平均より上で $y$ が平均より下なら、ずれの積は負になります。
  - こうした正負を全体でならすと、2 つの変量が同じ向きに動きやすいか、逆向きに動きやすいかが見えてきます。
  - 共分散は「関係を説明する」量であって、片方からもう片方を予測する式そのものではありません。
  - 単位を変えると値の大きさも変わるので、強さの比較だけなら相関係数のほうが向いています。
visual_explanation:
  - 散布図で点が右上がりに集まりやすければ、平均からのずれの積は正になりやすく、共分散は正になりやすいです。
  - 点が右下がりに並びやすければ、片方が平均より大きいときにもう片方が小さくなりやすく、共分散は負になりやすいです。
  - 点が四方にばらばらに散っていれば、正の積と負の積が打ち消し合い、共分散は 0 に近づきます。
where_it_appears:
  - 2 つの量の動き方を、散布図だけでなく数でも要約したいとき
  - 相関係数の式で、共通の動き方を標準化する前の材料として
  - 回帰直線の傾きがどちら向きになるかを考える前提として
practical_examples:
  - title: 勉強時間と得点
    description: 4 人の勉強時間を $x=(1,2,3,4)$ 時間、得点を $y=(50,55,70,85)$ 点とすると、勉強時間が平均より大きい人ほど得点も平均より大きい傾向があり、共分散は正になります。右上がりの散布図を数で要約した形です。
  - title: 気温と暖房使用時間
    description: 気温が高い日に暖房使用時間が短くなるなら、平均からのずれの向きは逆になりやすく、共分散は負になります。片方が増えるともう片方が減る傾向を表しています。
  - title: 単位を変えると大きさが変わる
    description: 身長を cm で測るか m で測るかを変えると、共分散の値の大きさも変わります。したがって、共分散の絶対値だけで関係の強さを比べるのは危険です。
exam_points:
  - 共分散の符号は、2 つの変量が同じ向きに動きやすいか、逆向きに動きやすいかを表します。
  - 共分散が正でも負でも、それだけで因果関係までは言えません。
  - 共分散の大きさは単位に依存するので、強さの比較にはそのまま使いにくいです。
  - 共分散は関係の要約であり、相関係数や回帰直線の前段階にある量です。
  - 散布図で形を確かめてから共分散の符号を読むと、意味を取り違えにくくなります。
formulas:
  - name: データの共分散
    latex: 's_{xy}=\frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})(y_i-\bar{y})'
    description: 各観測での平均からのずれを掛け合わせ、その平均をとると共分散になります。2 つの変量が一緒に動く向きを数で表した式です。
    conditions:
      - '$x_i$ と $y_i$ は同じ観測に対応する 2 つの値です。'
      - '$\bar{x}$ と $\bar{y}$ はそれぞれの平均です。'
      - ここでは手元のデータに対する共分散を、$n$ で割る形で表しています。
  - name: 確率変数の共分散
    latex: '\operatorname{Cov}(X,Y)=E[(X-E[X])(Y-E[Y])]'
    description: 確率変数 $X$ と $Y$ の平均からのずれの積の期待値として、共分散を定義する形です。データの共分散の考え方を確率変数へ広げたものです。
    conditions:
      - '$E[X]$ と $E[Y]$ はそれぞれの期待値です。'
      - 期待値が存在することを前提にします。
rigorous_explanation:
  - 共分散は、偏差 $(x_i-\bar{x})$ と $(y_i-\bar{y})$ の積を平均した量です。積が正になるのは 2 つの偏差が同符号のとき、負になるのは異符号のときです。
  - したがって、共分散の符号は一緒に増減しやすいかどうかを表しますが、単位の影響を受けるため、大きさの比較には注意が必要です。
  - $x$ を 100 倍すると共分散も 100 倍になるので、同じ関係でも測り方で値が変わります。この点が相関係数との大きな違いです。
  - 回帰直線の傾きは共分散と $x$ の分散から求まるので、共分散は予測式の向きを決める材料にはなりますが、共分散そのものが予測式ではありません。
common_mistakes:
  - 共分散の絶対値が大きいほど、必ず関係が強いと決めつけること
  - 共分散が正だから因果関係があると考えること
  - 共分散と相関係数を同じものとして扱うこと
  - 共分散が 0 に近いことを、どんな関係も存在しないことだと考えること

共分散は、散布図で見た「右上がり」「右下がり」を数でまとめた最初の一歩です。強さを単位の影響なしに比べたいときは `correlation-coefficient` に進み、片方からもう片方を予測したいときは回帰の考え方へ進みます。
