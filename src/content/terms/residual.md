short_definition: 各観測値 $y_i$ と回帰直線による予測値 $\hat{y}_i$ の差で、直線で説明しきれなかった部分です。
definition: 残差は、各観測について実際の値 $y_i$ から回帰直線による予測値 $\hat{y}_i$ を引いた $e_i=y_i-\hat{y}_i$ のことです。点が回帰直線より上にあれば残差は正、下にあれば負になります。残差は標本データに対して計算される観測後のずれであり、理論モデルに出てくる誤差項そのものと同一ではありません。回帰のあてはまりを判断するときは、残差の大きさや並び方を見ます。
intuition:
  - 残差は、回帰直線で予測したあとに「まだ残ったずれ」です。
  - 実際の点が直線より上なら正、下なら負になります。
  - 残差が全体として小さければ、回帰直線はデータによくあてはまっていると考えられます。
  - 残差が規則的な形で並ぶなら、直線モデルが関係の形を取り逃しているかもしれません。
  - 残差は回帰の量であって、相関係数のような対称な要約とは役割が違います。
visual_explanation:
  - 散布図の各点から回帰直線に下ろした縦の線分の長さが残差の大きさです。
  - 点が直線のすぐ近くに並ぶほど、残差は小さくなります。
  - 残差が上側にばかり続いたり、波のような形を描いたりするなら、単純な直線では十分でない可能性があります。
where_it_appears:
  - 最小二乗法で回帰直線を求めるとき
  - 決定係数や残差平方和で、回帰のあてはまりを評価するとき
  - 外れ値や曲線的な関係の有無を確認するとき
practical_examples:
  - title: 勉強時間と得点
    description: $x=(1,2,3,4)$、$y=(52,54,61,63)$ に対して回帰直線 $\hat{y}=47.5+4x$ を使うと、予測値は $(51.5,55.5,59.5,63.5)$ です。したがって残差は $(0.5,-1.5,1.5,-0.5)$ になります。
  - title: 売上の予測誤差
    description: ある日の売上の予測値が 100 個で、実際の売上が 110 個なら残差は $110-100=10$ です。予測より実際が上に外れたことを意味します。
  - title: 曲線的な関係の見落とし
    description: 点が U 字型に並んでいるのに直線を引くと、左端と右端で同じ符号の残差が続くことがあります。残差の並びを見ると、直線では表しにくい関係だと気づけます。
exam_points:
  - 残差は $e_i=y_i-\hat{y}_i$ で定義されます。
  - 点が回帰直線より上なら残差は正、下なら負です。
  - 最小二乗法は残差の二乗和を最小にする方法です。
  - 切片を含む最小二乗回帰では、残差の和は 0 になります。
  - 残差は回帰のあてはまりを見る量であり、理論上の誤差項とそのまま同一視しません。

rigorous_explanation:
  - 残差は、標本から推定した回帰直線に対する観測後のずれです。したがって、母集団モデルに出てくる誤差項 $\varepsilon$ と概念的には近いですが、同一ではありません。
  - 単回帰では $x$ から $y$ を予測するので、残差は縦方向のずれとして測ります。ここでも向きが決まっている点が、相関との違いです。
  - 最小二乗法では、個々の残差ではなく残差平方和を小さくします。そのため、平均的に小さなずれだけでなく、大きな外れも重く評価されます。
  - 残差の大きさだけでなく並び方も重要で、規則的なパターンがあれば直線モデルの見直しが必要なことがあります。
proof:
  summary: 切片を含む最小二乗回帰では、正規方程式から残差の総和が 0 になることが分かります。
  outline_steps:
    - '回帰直線を $\hat{y}_i=a+bx_i$ とすると、残差は $e_i=y_i-a-bx_i$ です。'
    - '最小二乗法の正規方程式の 1 つは、$\sum_{i=1}^{n}(y_i-a-bx_i)=0$ です。'
    - '左辺はちょうど $\sum_{i=1}^{n}e_i$ なので、$\sum_{i=1}^{n}e_i=0$ が得られます。'
    - 'したがって、切片を含む最小二乗回帰では、正の残差と負の残差が全体として釣り合います。'
common_mistakes:
  - 残差と理論上の誤差項を全く同じものだと思うこと
  - 残差が正なら、その観測が必ず異常だと決めつけること
  - 残差の平均が 0 なら、それだけで良いモデルだと考えること
  - 個々の残差が 0 でなければ回帰直線は使えないと考えること

`least-squares-method` は、この `residual` の二乗和を最小にするように回帰直線を決める方法です。さらに `coefficient-of-determination` は、残差として残った変動が全体の変動に対してどれくらい小さいかを割合で見ます。

残差は回帰の文脈で初めて意味を持つ量なので、相関係数のような対称な関係の要約とは別物です。相関が高いかどうかだけでは、どの点がどれだけ外れているかは分からないため、残差を見る価値があります。
