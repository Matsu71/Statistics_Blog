short_definition: 独立な 2 つのカイ二乗分布を自由度で割って比にした、0 以上の値をとる確率分布です。
definition: F分布は、独立な 2 つのカイ二乗分布 $U\sim\chi^2(\nu_1)$、$V\sim\chi^2(\nu_2)$ をそれぞれ自由度で割り、その比 $\frac{U/\nu_1}{V/\nu_2}$ で定義される確率分布です。値は必ず 0 以上で、自由度 $(\nu_1,\nu_2)$ の 2 つによって形が決まります。分散どうしの比較や、分散分析、回帰の有意性の検定で現れます。大きい値は、分子側のばらつきが分母側より相対的に大きいことを意味します。
intuition:
  - F分布は、2 つの「分散のような量」を比べるための分布だと考えると入りやすいです。
  - 比なので、分子と分母が同じくらいなら 1 付近、分子が大きければ 1 よりかなり大きくなります。
  - カイ二乗分布どうしの比なので、値は負になりません。
  - 自由度が小さいと右に長い裾を引き、自由度が大きいほど 1 の近くに集まりやすくなります。
  - 分子と分母を入れ替えると別の量になるので、向きがあります。
visual_explanation:
  - 横軸の 0 から始まり、右側へ長い裾を引く分布として描かれます。
  - 自由度が小さいと大きな F 値も出やすく、裾が重くなります。
  - 自由度が大きいと、比が 1 付近に集まりやすくなり、分布の山もそこに寄ります。
where_it_appears:
  - 2 つの分散を比較する検定
  - 分散分析で、群間の変動と群内の変動を比べるとき
  - 回帰直線が意味のある説明力を持つかを、説明された変動と残差変動の比でみるとき
practical_examples:
  - title: 比が 1 付近のとき
    description: たとえば $U=10$、$\nu_1=5$、$V=12$、$\nu_2=6$ なら、$F=\frac{10/5}{12/6}=1$ です。分子側と分母側の分散の大きさが、自由度調整後には同程度だと読めます。
  - title: 回帰での F 値
    description: 回帰で、説明された平均平方が残差平均平方の 9 倍なら $F=9$ です。これは、直線で説明できた変動が残差による変動よりかなり大きいことを示します。
  - title: 分子と分母を入れ替えると変わる
    description: 同じ 2 つの量でも、どちらを分子に置くかで F 値は逆数方向に変わります。F分布は比の向きを持つので、式の並びを取り違えないことが大切です。
exam_points:
  - F分布は、独立な 2 つのカイ二乗分布の比から作られます。
  - 値は 0 以上で、自由度は 2 つ必要です。
  - 1 付近なら両者のばらつきが近く、大きい値ほど分子側が相対的に大きいと読みます。
  - 回帰や分散分析では、説明された変動と残差変動の比として F 値が現れます。
  - F 値が大きくても、それだけで因果関係が示されるわけではありません。
formulas:
  - name: F分布の定義
    latex: 'F=\frac{U/\nu_1}{V/\nu_2}'
    description: 独立な 2 つのカイ二乗変数を、それぞれ自由度で割って比にしたものが F 分布です。
    conditions:
      - '$U\sim\chi^2(\nu_1)$、$V\sim\chi^2(\nu_2)$ は独立です。'
      - '$\nu_1,\nu_2$ は自由度です。'
  - name: F分布の表記
    latex: 'F\sim F(\nu_1,\nu_2)'
    description: F分布は 2 つの自由度で指定します。1 つでは形が決まりません。
    conditions:
      - 第 1 自由度は分子側、第 2 自由度は分母側に対応します。
  - name: 回帰での F 値
    latex: 'F=\frac{\text{回帰による平均平方}}{\text{残差平均平方}}'
    description: 回帰では、説明された変動を自由度で割った量と、残差による変動を自由度で割った量の比として F 値を考えます。回帰直線が有効かどうかをみる指標の 1 つです。
    conditions:
      - 分子と分母の両方で自由度調整を行います。
      - 回帰全体の有意性を見るときに使われます。
rigorous_explanation:
  - F分布はカイ二乗分布の比でできているので、カイ二乗分布と同様に 0 以上の値しか取りません。
  - 比の分子と分母には役割の違いがあるため、自由度も $(\nu_1,\nu_2)$ と 2 つ必要です。順序を入れ替えると別の分布になります。
  - 回帰では、説明された変動が残差変動に比べて十分大きいかを F 値で見ます。したがって、回帰直線がどれだけ意味のある傾向を持つかを判断する道具として使われます。
  - ただし、F 値が大きいことは「その標本で説明された変動が相対的に大きい」ことを示すだけで、因果関係や実務上の重要性を直接保証するものではありません。
proof:
  summary: F分布の基本的な性質は、独立なカイ二乗変数の比という定義から確認できます。
  outline_steps:
    - '$U$ と $V$ はカイ二乗変数なので、どちらも 0 以上です。'
    - '自由度 $\nu_1,\nu_2$ で割っても正の量のままなので、$\frac{U/\nu_1}{V/\nu_2}$ も 0 以上です。'
    - '分子と分母の自由度が別なので、分布の形を決めるには $(\nu_1,\nu_2)$ の 2 つが必要です。'
    - 'もし逆数をとれば、$\frac{1}{F}=\frac{V/\nu_2}{U/\nu_1}$ となり、分子と分母の役割が入れ替わります。'
    - 'このことから、F分布は比の向きと 2 つの自由度を持つ分布だと分かります。'
common_mistakes:
  - F 値が負になることもあると思うこと
  - F分布の自由度を 1 つだけで決まると考えること
  - 分子と分母を入れ替えても同じ F 値だと思うこと
  - F 検定で有意だったことを、そのまま因果関係の証明だと考えること

`regression-line` は、説明変数から目的変数を予測する直線そのものを扱います。`f-distribution` は、その回帰直線が残差に比べてどれくらい意味のある説明力を持っているかを、変動の比として評価するときに現れる分布です。

カイ二乗分布を材料にした比の分布なので、まずは「二乗したずれを足し合わせる量」と「自由度」の感覚を押さえることが大切です。そのうえで回帰に戻ると、説明された変動と残った変動を比べるという読み方がしやすくなります。
