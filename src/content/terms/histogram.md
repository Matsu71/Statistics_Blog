short_definition: データを階級に分け、各階級の度数を柱の高さで表すグラフです。
definition: ヒストグラムは、連続的な数値データをいくつかの階級に分け、それぞれの階級に入るデータ数を柱で表した図です。分布の山、偏り、散らばり、外れた値の有無を視覚的に確認できます。
intuition:
  - ヒストグラムは、数値データを幅ごとの箱に分けて、どの箱に何個入ったかを見る図です。
  - 棒が高いところは、その範囲にデータが多いことを表します。
  - データの中心だけでなく、分布の形や左右の偏りを見やすくします。
  - 箱ひげ図が要約を短く見せる図なら、ヒストグラムは分布の形をより直接見せる図です。
  - ただし、階級にまとめるため、元の個別の値そのものは図から完全には戻せません。
visual_explanation:
  - 0 点台、10 点台、20 点台のように階級を作り、それぞれに何人いるかを柱で表します。
  - 柱が中央付近に集まれば中心に多い分布、片側に長く伸びれば偏りのある分布として読めます。
  - 階級幅を変えると見え方も変わるため、問題文で指定された階級をよく確認します。
where_it_appears:
  - テスト点数、身長、待ち時間など、数値データの分布を見るとき
  - 正規分布に近い形か、左右に偏っているかをざっくり確認するとき
  - 箱ひげ図と合わせて、分布の形と要約を比較するとき
practical_examples:
  - title: 点数分布を見る
    description: テスト点数を 0-9 点、10-19 点、20-29 点のように分けると、どの点数帯に人数が多いかを柱の高さで見られます。
  - title: 階級幅で見え方が変わる
    description: 5 点刻みと 20 点刻みでは、同じデータでも見え方が変わります。細かすぎるとばらばらに、粗すぎると形がつぶれて見えることがあります。
  - title: 分布の偏りを読む
    description: 待ち時間のヒストグラムで右側に長い尾が伸びていれば、たまに非常に長く待つ人がいると読めます。平均だけでは見えにくい偏りを確認できます。
exam_points:
  - 横軸は階級、縦軸は度数または相対度数を表します。
  - 棒グラフと違い、数値の範囲を連続した階級として扱うのが基本です。
  - 階級幅、階級の境界、度数を確認して読みます。
  - 分布の山、偏り、外れた値の有無を読み取る練習が重要です。
  - 階級幅がそろっていない図では、単純に柱の高さだけで比較してよいかを確認します。
formulas:
  - name: 相対度数
    latex: '\text{relative frequency}=\frac{\text{class frequency}}{n}'
    description: 各階級の度数をデータ数 $n$ で割ると、全体に占める割合として読めます。
    conditions:
      - '$n$ はデータ数です。'
      - class frequency は、その階級に入るデータ数です。
  - name: 階級幅の目安
    latex: 'h=\frac{x_{\max}-x_{\min}}{k}'
    description: 最大値と最小値の幅を階級数 $k$ で割ると、階級幅の目安になります。実際には読みやすい区切りに調整します。
    conditions:
      - '$h$ は階級幅の目安です。'
      - '$k$ は階級数です。'
      - '$x_{\max}$ は最大値、$x_{\min}$ は最小値です。'
rigorous_explanation:
  - ヒストグラムは、個々の値を階級にまとめて分布の形を見るための図です。
  - 階級幅が同じなら、柱の高さを比べることで度数の多い範囲が分かります。
  - 階級の取り方によって見え方が変わるため、図から結論を読むときは階級幅と境界を確認します。
  - ヒストグラムは分布の形を見る図であり、個々の観測値の順番や全ての正確な値を保存する図ではありません。
common_mistakes:
  - 棒グラフと同じものだと思うこと
  - 階級幅を確認せずに柱の高さだけを比べること
  - ヒストグラムから平均や中央値が正確に分かると思い込むこと
  - 階級の境界にある値をどちらへ入れるか確認しないこと

ヒストグラムは、データの形を眺めるための入口です。分布の山、偏り、広がりを図で見られるので、平均や標準偏差だけでは分かりにくい情報を補えます。

`box-plot` は分布を短く要約する図です。ヒストグラムで形を見て、箱ひげ図で代表的な位置と広がりを確認すると、データの読み取りが安定します。
