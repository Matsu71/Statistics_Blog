short_definition: 回帰直線を引くときに、観測値と予測値のずれの二乗和が最小になるよう係数を決める方法です。
definition: 最小二乗法は、各観測値 $y_i$ と回帰直線による予測値 $\hat{y}_i$ の差を残差とし、その二乗和 $\sum (y_i-\hat{y}_i)^2$ が最も小さくなるように直線の係数を選ぶ方法です。残差をそのまま足すと正負が打ち消し合うので、二乗してから合計するところが特徴です。単回帰分析で回帰直線を求める標準的な方法ですが、「最良」の意味はあくまでこの二乗和という基準に関してです。
intuition:
  - 各点と回帰直線のずれが全体として小さくなるように、線の位置と傾きを決める方法です。
  - 残差をそのまま足すと、上にはみ出した点と下にはみ出した点が打ち消し合ってしまいます。
  - 二乗すると符号が消え、ずれが大きい点ほど強く効くようになります。
  - 単回帰では、横方向ではなく縦方向のずれを小さくするのが基本です。これは $x$ から $y$ を予測する向きを決めているからです。
  - 最小二乗法は直線を選ぶルールであって、関係が因果的かどうかを判断するルールではありません。
visual_explanation:
  - 各点から回帰直線へ下ろした縦線の長さが残差で、その長さの二乗を全部足したものを最小にするイメージです。
  - ある 1 点が大きく外れていると、その二乗が強く効くので、回帰直線はその点にもある程度引っ張られます。
  - 右上がりすぎる線や右下がりすぎる線では、どこかの点の縦のずれが大きくなり、二乗和も大きくなります。
where_it_appears:
  - 回帰直線の係数を実際に計算するとき
  - 残差平方和、決定係数、回帰のあてはまりを理解するとき
  - 単回帰分析の計算問題で、傾きや切片の公式を導く場面
practical_examples:
  - title: 勉強時間と得点
    description: $x=(1,2,3,4)$、$y=(52,54,61,63)$ に対して最小二乗法で求めると、回帰直線は $\hat{y}=47.5+4x$ になります。このとき残差は $(0.5,-1.5,1.5,-0.5)$、残差平方和は $0.25+2.25+2.25+0.25=5$ です。
  - title: 打ち消し合いを防ぐ
    description: 残差が $+3$ と $-3$ なら、和は 0 ですが、ずれがないわけではありません。二乗すれば $9+9=18$ となり、ずれがしっかり評価されます。
  - title: 外れた点の影響
    description: ほとんどの点が直線の近くにあっても、1 点だけ大きく離れていると、その点の二乗が大きくなるので回帰直線は影響を受けやすくなります。
exam_points:
  - 最小二乗法は、残差平方和を最小にする方法です。
  - 残差をそのまま足すのではなく、二乗してから足すのは打ち消し合いを防ぐためです。
  - 単回帰では、$x$ から $y$ を予測するので縦方向のずれを小さくします。
  - 最小二乗法で求めた回帰直線の傾きは $b=\frac{s_{xy}}{s_x^2}$、切片は $a=\bar{y}-b\bar{x}$ です。
  - 最小二乗法は二乗和という基準で最良なだけで、因果関係や将来予測の正しさを保証するものではありません。
formulas:
  - name: 残差平方和
    latex: 'S(a,b)=\sum_{i=1}^{n}(y_i-a-bx_i)^2'
    description: 直線 $\hat{y}=a+bx$ に対して、各観測値との縦方向のずれを二乗して足し合わせたものです。最小二乗法では、この $S(a,b)$ を最小にする $a,b$ を探します。
    conditions:
      - '$y_i$ は観測値、$a+bx_i$ はその予測値です。'
      - 二乗しているので、正負の打ち消し合いは起こりません。
  - name: 傾き
    latex: 'b=\frac{s_{xy}}{s_x^2}'
    description: 最小二乗法で求めた単回帰の傾きです。$x$ と $y$ の共分散を、$x$ の分散で割った形になります。
    conditions:
      - '$s_{xy}$ は共分散、$s_x^2$ は $x$ の分散です。'
      - '$s_x^2>0$、つまり $x$ が全て同じ値ではないことを前提にします。'
  - name: 切片
    latex: 'a=\bar{y}-b\bar{x}'
    description: 傾きが決まると、回帰直線が平均点 $(\bar{x},\bar{y})$ を通ることから切片が求まります。
    conditions:
      - '$\bar{x}$ と $\bar{y}$ はそれぞれの平均です。'
rigorous_explanation:
  - 最小二乗法では、残差平方和 $S(a,b)$ を最小にすることで、全体として縦方向のずれがもっとも小さい直線を選びます。
  - 残差の和 $\sum (y_i-\hat{y}_i)$ は、切片を含む回帰では 0 になりやすく、ずれの大きさそのものを測る基準としては使いにくいです。そのため二乗和を使います。
  - 傾きの公式 $b=\frac{s_{xy}}{s_x^2}$ は、$x$ のばらつきに対して $y$ がどれくらい一緒に動くかを表した形です。
  - 最小二乗法で決めた直線は、外れ値の影響を受けやすい面があります。二乗が大きいずれを強く重く扱うからです。
proof:
  summary: 残差平方和を係数で微分して 0 とおくと、最小二乗法の公式が得られます。
  outline_steps:
    - '残差平方和 $S(a,b)=\sum_{i=1}^{n}(y_i-a-bx_i)^2$ を考えます。'
    - '$a$ で偏微分して 0 とおくと、$\sum_{i=1}^{n}(y_i-a-bx_i)=0$ です。'
    - '$b$ で偏微分して 0 とおくと、$\sum_{i=1}^{n}x_i(y_i-a-bx_i)=0$ です。'
    - 'これら 2 本の正規方程式を解くと、$a=\bar{y}-b\bar{x}$ と $b=\frac{s_{xy}}{s_x^2}$ が得られます。'
    - 'したがって、最小二乗法は残差平方和を最小にする回帰直線を具体的に計算できる方法です。'
common_mistakes:
  - 残差の和を最小にすればよいと考えること
  - 横方向のずれや点から直線への最短距離を最小にしていると誤解すること
  - 最小二乗法で求めた直線なら、全ての点をほぼ必ず通ると思うこと
  - 最小二乗法で直線が決まったことを、そのまま因果関係の証拠だと考えること

`simple-linear-regression` では、平均的な関係を直線モデルで表します。そのとき具体的にどの直線を採用するかを決めるルールが `least-squares-method` です。そこで最小化している対象が `residual` の二乗和だと分かると、計算と意味づけがつながります。

相関係数は 2 変量の結びつきを要約するだけですが、最小二乗法は「$x$ から $y$ を予測する直線をどう決めるか」という回帰の計算手順です。ここでも、相関と回帰の役割は別だと整理して読むことが大切です。
