short_definition: 分散を元の単位に戻して解釈しやすくした、散らばりの代表的な指標です。
definition: 分散の平方根をとった値を標準偏差といいます。分散は二乗の単位になるため、標準偏差にすると元のデータの単位で散らばりを見られます。平均のまわりにどれくらい値が集まりやすいかを、直感的に読みやすくした指標です。ただし、標準偏差は「平均との差の単純な平均」ではなく、二乗したずれの平均に平方根をとった量です。
intuition:
  - 標準偏差は、平均からどのくらい離れやすいかを元の単位のまま見るための指標です。
  - 分散よりも、現実の数字としてイメージしやすいという利点があります。
  - 「だいたい平均から何くらい離れるか」の感覚をつかむのに向いています。
  - 平均が同じでも標準偏差が違えば、値のまとまり方が違うと分かります。
  - 平均が中心の位置を表すのに対して、標準偏差はその中心のまわりで値がどれくらい広がるかを表します。
  - 分散より現実の単位で読みやすい一方、外れ値の影響を受ける点は分散と同じです。
visual_explanation:
  - 平均点が同じ 2 クラスでも、点数が広く散っているクラスのほうが標準偏差は大きくなります。
  - 標準偏差が小さいと、平均の近くにデータが集まりやすいと考えられます。
  - 分散をそのまま見せるより、標準偏差のほうが値の大きさを直感で読みやすいです。
  - 分散が「点数の二乗」の単位になるなら、平方根をとった標準偏差はふたたび「点」の単位で読めます。
  - 同じ平均でも、標準偏差が大きい図は平均から左右へ広く伸びた形として見えます。
where_it_appears:
  - データのばらつきを、元の単位のまま説明したいとき
  - 正規分布、標準化、推定の話題に入る前の基本指標として
  - 平均だけでなく散らばりもセットで説明したい場面

  - 標準偏差が小さいほど、平均のまわりに値が集まりやすいと読みます。

formulas:
  - name: 母標準偏差
    latex: '\sigma = \sqrt{\sigma^2}'
    description: 母分散の平方根をとると母標準偏差になります。母分散が元データの単位の二乗で表されるため、平方根をとると元の単位に戻ります。
    conditions:
      - '$\sigma^2$ は母分散です。'
      - '$\sigma$ は母標準偏差です。'
      - 母集団全体の散らばりを元の単位で読むための量です。
  - name: 標本標準偏差の考え方
    latex: 's = \sqrt{s^2}'
    description: 不偏分散や標本分散の平方根として標本標準偏差を考えます。どの分散を平方根にしているかで意味が変わるため、問題文の定義を確認します。
    conditions:
      - 何を平方根にしているかを常に確認します。
      - '$s^2$ は文脈によって標本分散や不偏分散を表すことがあります。'
  - name: データの標準偏差
    latex: 's_n = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})^2}'
    description: 観測値 $n$ 個の平均との差の二乗平均に平方根をとると、手元のデータの標準偏差になります。
    conditions:
      - '$x_i$ は i 番目の観測値です。'
      - '$\bar{x}$ は平均です。'
      - ここでは $n$ 個で割る分散の平方根として表しています。
rigorous_explanation:
  - 分散は単位が二乗になるため、その平方根である標準偏差のほうが解釈しやすくなります。
  - 標準偏差は、平均からの典型的な離れ方を元の単位で表そうとする指標です。
  - 標準偏差は平均との差の絶対値を単純に平均した量ではありません。二乗して平均し、そのあと平方根をとるため、平均から大きく離れた値の影響も受けます。
  - 標準偏差が 0 なら、分散も 0 なので、すべての値が平均と一致しています。
  - 標本平均や正規分布を学ぶときにも、散らばりを表す基準として何度も使われます。
proof:
  summary: 標準偏差は、分散の単位を元に戻すために平方根をとった量として理解できます。
  outline_steps:
    - '分散は $\frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})^2$ のように、平均との差を二乗してから平均します。'
    - 'たとえば元データの単位が点なら、分散の単位は点の二乗になります。'
    - '平方根をとると単位が元に戻り、$s_n=\sqrt{\frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})^2}$ と読めます。'
    - 'そのため標準偏差は、分散と同じ散らばりを、元データと同じ単位で読みやすくした指標です。'
common_mistakes:
  - 標準偏差と分散の違いを、単位の違いまで含めて説明しないこと
  - 標準偏差が大きいことを平均が大きいことと混同すること
  - 標準偏差が小さいと、必ず平均に近い値しか出ないと考えてしまうこと
  - 標準偏差を、平均との差の絶対値の平均と同じものだと思い込むこと
  - 標本標準偏差で、どの分散の平方根を使っているかを確認しないこと

標準偏差は、`variance` の考え方を実際の数値感覚へ戻す役割を持ちます。分散との違いを「平方根をとって単位を戻したもの」と言葉で説明できるようにしておくと、試験でも実務でも役立ちます。`normal-distribution` を読むときも、平均とあわせて標準偏差を見る流れが基本です。

平均は中心、標準偏差は中心のまわりの広がりです。この 2 つをセットで読むと、「どこに集まっているか」と「どれくらい散っているか」を分けて説明できます。
