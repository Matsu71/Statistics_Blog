short_definition: 帰無仮説が正しいのに棄却してしまう危険を、検定の前にどこまで許すかを表す基準値です。
definition: 有意水準とは、仮説検定で帰無仮説 $H_0$ が正しいのに棄却してしまう危険をどこまで許すかを、あらかじめ決めておく基準値です。通常は $\alpha$ と書き、0.05 や 0.01 がよく使われます。有意水準はデータを見る前に設定するもので、観測後に計算する p値とは役割が異なります。有意水準を小さくすると棄却しにくくなり、第一種の過誤を起こしにくくなる一方で、差を見逃しやすくなることがあります。
intuition:
  - 有意水準は、「どれくらい厳しく棄却するか」を先に決めるルールです。
  - 0.05 なら、帰無仮説のもとでめったに起きない 5% 程度の領域を棄却域にします。
  - 0.01 に下げると、もっと珍しい結果だけを棄却の対象にします。
  - 有意水準は標本ごとに変わる値ではなく、検定設計の値です。
  - 厳しくしすぎると、今度は本当にある差も見逃しやすくなります。
visual_explanation:
  - 帰無仮説の分布の端に、全体で 5% や 1% の面積を塗って棄却域を作る図を思い浮かべると分かりやすいです。
  - 有意水準が小さいほど、塗られる領域は分布のさらに外側へ移ります。
  - 片側検定ならその面積を片側に、両側検定なら左右に分けて置きます。
  - p値は観測後に計算する面積で、有意水準は先に決める面積という違いがあります。
where_it_appears:
  - 仮説検定で棄却域を決める場面
  - p値と比較して棄却するかどうかを判定する場面
  - 第一種の過誤との関係を問う試験問題
  - 片側検定と両側検定で臨界値が変わる理由を説明する場面
practical_examples:
  - title: 0.05 と 0.01 の違い
    description: 同じデータで p値が 0.03 でも、有意水準 0.05 なら棄却し、有意水準 0.01 なら棄却しません。有意水準が厳しいほど、棄却のハードルは高くなります。
  - title: 品質管理の警報基準
    description: 工場で異常を見逃したくないが、誤警報も多すぎると困るとき、どこで警報を出すかを先に決める感覚が有意水準に近いです。検査結果が出てから基準を動かしてはいけません。
  - title: 片側と両側で使い方が違う
    description: 有意水準 5% でも、右片側検定では右側 5% を全部使い、両側検定では左右 2.5% ずつに分けます。同じ 5% でも棄却域の形は変わります。
exam_points:
  - 有意水準 $\alpha$ は、データを見る前に決める基準です。
  - 有意水準は p値ではなく、p値と比較される側の値です。
  - 有意水準を小さくすると棄却しにくくなり、第一種の過誤を抑えやすくなります。
  - 同じ有意水準でも、片側検定と両側検定で棄却域の位置は変わります。
  - 第一種の過誤を抑えるほど、第二種の過誤が増えやすいことがあります。
formulas:
  - name: 有意水準の基本条件
    latex: 'P_{H_0}(T\in R)\le \alpha'
    description: 棄却域 $R$ は、帰無仮説が正しいときに検定統計量 $T$ が入り込む確率が有意水準 $\alpha$ 以下になるように決めます。
    conditions:
      - '$R$ は棄却域です。'
      - '$T$ は検定統計量です。'
      - 離散分布の検定などでは、ちょうど $\alpha$ ではなく「以下」になることがあります。
  - name: p値による判定規則
    latex: 'p\le \alpha \Rightarrow \text{reject } H_0'
    description: 観測データから計算した p値 $p$ が有意水準 $\alpha$ 以下なら、帰無仮説を棄却します。
    conditions:
      - '$p$ は観測後に計算する p値です。'
      - '$\alpha$ は観測前に決めておく有意水準です。'
      - 片側検定か両側検定かに応じて p値の計算方法は変わります。
rigorous_explanation:
  - 有意水準は、検定手続きそのものに属する設計値です。したがって、同じ検定手順を繰り返したときに第一種の過誤がどれくらい起こりうるかを制御する役割を持ちます。
  - 連続分布を使う基本的な検定では、棄却域の確率をちょうど $\alpha$ に合わせることが多いですが、離散分布では「$\alpha$ 以下」にしかできないことがあります。
  - 有意水準を小さくすると、帰無仮説を棄却するために必要な証拠はより強くなります。その代わり、真の差があるときにも棄却できない第二種の過誤が増えやすくなります。
  - 有意水準は研究や業務の目的に応じて決めますが、標本を見たあとに変更すると検定の意味が崩れます。
proof:
  summary: 有意水準は、帰無仮説のもとで棄却域に入る確率を上から抑えるように、観測前に設定されます。
  outline_steps:
    - 'まず、帰無仮説のもとでの検定統計量 $T$ の分布を考えます。'
    - 'その分布の端から、合計で $\alpha$ 以下の確率になるように棄却域 $R$ を決めます。'
    - '観測後は、実際の統計量が $R$ に入ったか、または p値が $\alpha$ 以下かで判定します。'
    - 'したがって、有意水準は観測後に変わる値ではなく、観測前に固定する判定基準だと分かります。'
common_mistakes:
  - 有意水準を「帰無仮説が真である確率」だと思うこと
  - 有意水準と p値を同じものとして扱うこと
  - 標本を見たあとで有意水準を都合よく変えること
  - 有意水準を小さくすれば、いつでも検定が良くなると思うこと

`p-value` と対で理解すると、有意水準は「先に決める基準」、p値は「あとで計算する結果」だとはっきりします。ここを取り違えないことが、仮説検定の読み間違いを減らす近道です。

`rejection-region` は、この有意水準を図の上で表したものです。有意水準が決まると、どの範囲を棄却域にするかが決まり、そこに観測値が入るかどうかで判定できます。
