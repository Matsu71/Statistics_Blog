short_definition: 切片を含む単回帰で、回帰直線が目的変数の変動をどの程度とらえられているかを 0 から 1 の範囲で表す指標です。
definition: 決定係数は、目的変数 $y$ の全体の変動のうち、回帰直線で表せた部分の割合を表す指標で、通常 $R^2$ と書きます。切片を含む単回帰では 0 から 1 の範囲をとり、1 に近いほど標本上で直線のあてはまりがよいと読めます。ただし、決定係数が高いからといって因果関係が示されたとはいえず、将来の予測が必ず正確だとも限りません。決定係数はあくまで、その標本での直線モデルのあてはまりを要約する量です。
intuition:
  - まず $y$ のばらつき全体を考え、そのうちどれだけが回帰直線で説明できたかを割合で見る指標です。
  - 回帰直線の近くに点が集まっていて残差が小さいほど、決定係数は大きくなります。
  - 回帰直線が平均 $\bar{y}$ だけを使う予測よりほとんど改善していなければ、決定係数は小さくなります。
  - 切片を含む単回帰では、決定係数は相関係数 $r$ の二乗に一致しますが、これは条件つきの関係です。
  - 決定係数が大きくても、「$x$ が $y$ を決めている」とまでは言えません。
visual_explanation:
  - 点が回帰直線の近くに細く並ぶ散布図では、残差が小さく、決定係数は大きくなりやすいです。
  - 点が広く散っていて、回帰直線を引いても平均線 $\bar{y}$ と大差なければ、決定係数は小さくなります。
  - 外れ値や曲線的な関係があると、決定係数の数値だけでは見えにくいので、散布図や残差も合わせて確認します。
where_it_appears:
  - 単回帰分析で、回帰直線のあてはまりのよさをまとめたいとき
  - 残差平方和と全体の変動を比較する問題
  - 相関係数 $r$ と $R^2$ の関係を問う問題
practical_examples:
  - title: 勉強時間と得点
    description: $x=(1,2,3,4)$、$y=(52,54,61,63)$ に対して回帰直線 $\hat{y}=47.5+4x$ を引くと、全体の変動 $\sum (y_i-\bar{y})^2=85$、残差平方和 $\sum e_i^2=5$ なので、$R^2=1-\frac{5}{85}\approx 0.94$ です。この標本では、得点の変動の約 94% を直線で表せていると読みます。
  - title: 高い決定係数でも因果とは限らない
    description: 気温の高い日にアイスの売上も水難事故件数も増えると、両者の回帰で高い $R^2$ が出ることがあります。しかし、アイスの売上が事故の原因だといえるわけではありません。
  - title: 低い決定係数でも無意味とは限らない
    description: 人の行動を含むデータでは個人差が大きいため、$R^2$ がそれほど高くなくても傾き自体に意味があることがあります。数値が低いだけで即座に「何も分からない」とは言い切れません。
exam_points:
  - 決定係数は、回帰直線のあてはまりのよさを割合で表す指標です。
  - 切片を含む単回帰では、$R^2=1-\frac{\mathrm{SSE}}{\mathrm{SST}}$ と書けます。
  - $R^2$ が高いことは、標本上で残差が小さいことを示しますが、因果関係を示すものではありません。
  - $R^2$ が低くても、関係が曲線的だったり、個人差が大きかったりする可能性があります。
  - 切片を含む単回帰では、決定係数は相関係数の二乗 $r^2$ に一致します。
formulas:
  - name: 残差から表した決定係数
    latex: 'R^2=1-\frac{\sum_{i=1}^{n}e_i^2}{\sum_{i=1}^{n}(y_i-\bar{y})^2}'
    description: 全体の変動 $\sum (y_i-\bar{y})^2$ のうち、残差平方和 $\sum e_i^2$ として残った割合を引くと、回帰直線で表せた割合になります。
    conditions:
      - '$e_i=y_i-\hat{y}_i$ は残差です。'
      - 切片を含む回帰を考えています。
      - 分母は $y$ の全体の変動を表します。
  - name: 回帰で表せた変動の割合
    latex: 'R^2=\frac{\sum_{i=1}^{n}(\hat{y}_i-\bar{y})^2}{\sum_{i=1}^{n}(y_i-\bar{y})^2}'
    description: 予測値 $\hat{y}_i$ が平均 $\bar{y}$ からどれだけ動いているかを見ると、回帰直線で表せた変動の割合として決定係数を読めます。
    conditions:
      - 分子は回帰で表せた変動、分母は全変動です。
      - 切片を含む最小二乗回帰では、この式と $1-\frac{\mathrm{SSE}}{\mathrm{SST}}$ が一致します。
  - name: 単回帰での相関との関係
    latex: 'R^2=r^2'
    description: 切片を含む単回帰では、決定係数は相関係数の二乗に一致します。したがって、相関の強い標本ほど $R^2$ も大きくなりやすいです。
    conditions:
      - 単回帰かつ切片を含む場合の関係です。
      - 相関係数 $r$ の符号は消え、強さだけが残ります。
rigorous_explanation:
  - 決定係数は、平均だけで予測する基準モデルと、回帰直線で予測するモデルを比べたときの改善度を割合で表していると見ると分かりやすいです。
  - 残差平方和 $\mathrm{SSE}$ が小さいほど、回帰直線で説明しきれなかった部分が少ないので、$R^2$ は大きくなります。
  - 切片を含む最小二乗回帰では、全変動 $\mathrm{SST}$ が「回帰で表せた変動」と「残差として残った変動」に分解されます。この分解が決定係数の基礎です。
  - 高い $R^2$ は、標本内で直線がよくあてはまっていることを示しますが、説明変数が目的変数の原因だという主張や、観測範囲外での予測精度までは保証しません。
proof:
  summary: 切片を含む最小二乗回帰では、全変動が回帰で表せた変動と残差変動に分解されるため、決定係数の式が得られます。
  outline_steps:
    - '各観測について、$y_i-\bar{y}=(\hat{y}_i-\bar{y})+e_i$ と分けられます。'
    - '両辺を二乗して和をとると、$\sum (y_i-\bar{y})^2=\sum (\hat{y}_i-\bar{y})^2+\sum e_i^2+2\sum e_i(\hat{y}_i-\bar{y})$ です。'
    - '切片を含む最小二乗回帰では、残差は予測値と直交するので、交差項 $\sum e_i(\hat{y}_i-\bar{y})$ は 0 になります。'
    - 'したがって、$\mathrm{SST}=\mathrm{SSR}+\mathrm{SSE}$ が得られます。'
    - '両辺を $\mathrm{SST}$ で割ると、$R^2=\frac{\mathrm{SSR}}{\mathrm{SST}}=1-\frac{\mathrm{SSE}}{\mathrm{SST}}$ となります。'
common_mistakes:
  - $R^2=0.8$ を「$x$ が $y$ の 80% の原因だ」と読んでしまうこと
  - 決定係数が高ければ、将来の予測も必ず高精度だと考えること
  - 決定係数が低ければ、関係が全くないと決めつけること
  - 散布図や残差を見ずに、$R^2$ の数値だけでモデルを判断すること

`residual` は、回帰直線で説明しきれずに残った部分です。`coefficient-of-determination` は、その残りが全体の変動に対してどれくらい小さいかを割合で見た指標だと考えると、式の意味がつかみやすくなります。

切片を含む単回帰では $R^2=r^2$ という関係がありますが、これは相関係数と回帰が同じものだという意味ではありません。相関は直線的な結びつきの要約、決定係数は回帰直線のあてはまりの要約であり、どちらも因果関係そのものを示す量ではありません。
