short_definition: 四分位数を使って、データの中心と広がりを短く表す図です。
definition: 箱ひげ図は、中央値、第 1 四分位数、第 3 四分位数、最小値、最大値などを使って、データの分布を要約する図です。箱は中央 50% の広がりを、箱の中の線は中央値を、ひげは端のほうの広がりを表します。
intuition:
  - 箱ひげ図は、データの形を「箱」と「ひげ」で短くまとめた地図です。
  - 箱が大きいほど、中央 50% のデータが広く散らばっています。
  - 中央線が箱の片側に寄っていれば、分布の偏りを考える手がかりになります。
  - ヒストグラムが分布の形を細かく見る図なら、箱ひげ図は複数のデータ群を比べやすい要約図です。
visual_explanation:
  - 箱の左端または下端が $Q_1$、箱の中の線が中央値 $Q_2$、箱の右端または上端が $Q_3$ です。
  - 箱の幅は四分位範囲 $Q_3-Q_1$ に対応します。
  - ひげの長さを見ると、中央から端へ向かう広がりを確認できます。
where_it_appears:
  - クラス別、地域別、月別など、複数グループの分布を比較するとき
  - 中央値と四分位範囲を同時に見たいとき
  - 外れ値や偏りの有無をざっくり確認したいとき
practical_examples:
  - title: 2 クラスの点数を比べる
    description: A クラスと B クラスの箱ひげ図を並べると、中央値が高いのはどちらか、中央 50% のばらつきが大きいのはどちらかを一目で比べられます。
  - title: 箱の大きさを読む
    description: 箱が小さいグループは、中央付近の点数が狭い範囲に集まっています。箱が大きいグループは、中央付近でもばらつきが大きいと読めます。
exam_points:
  - 箱の端は $Q_1$ と $Q_3$、箱の中の線は中央値です。
  - 箱の大きさは四分位範囲を表します。
  - 平均ではなく、中央値と四分位数を中心に読む図です。
  - 複数グループの分布比較で、中央値、散らばり、偏りを分けて見ます。
formulas:
  - name: 箱の幅
    latex: '\mathrm{IQR}=Q_3-Q_1'
    description: 箱ひげ図の箱の部分は、四分位範囲に対応します。
    conditions:
      - '$Q_1$ は第 1 四分位数です。'
      - '$Q_3$ は第 3 四分位数です。'
  - name: 5 数要約
    latex: '\min x_i,\ Q_1,\ Q_2,\ Q_3,\ \max x_i'
    description: 箱ひげ図は、最小値、四分位数、最大値を使ってデータを短く要約します。
    conditions:
      - '$Q_2$ は中央値です。'
      - 外れ値を別表示する流儀では、ひげの端の意味が変わることがあります。
rigorous_explanation:
  - 箱ひげ図は、順位に基づく要約を図にしたものです。
  - 箱は中央 50% のデータを表すため、端の外れ値に左右されにくい比較ができます。
  - ただし、箱ひげ図だけでは山の数や細かな形は分かりません。分布の形そのものを見たいときはヒストグラムも確認します。
common_mistakes:
  - 箱の中の線を平均だと思うこと
  - 箱の端を最小値・最大値だと勘違いすること
  - 箱が大きいほど平均が高いと読むこと
  - 箱ひげ図だけで分布の細かな山の形まで分かると思うこと

箱ひげ図は、`quartile` と `interquartile-range` を図として読むためのページです。平均ではなく、中央値と四分位数を中心に見る点が大切です。

複数のデータ群を比べるときは、箱ひげ図で要約を確認し、必要に応じてヒストグラムで分布の形を補って読むと、見落としが少なくなります。
