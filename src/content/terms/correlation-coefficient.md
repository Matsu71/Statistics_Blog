short_definition: 2つの変量の直線的な関係の向きと強さを、-1 から 1 の範囲で表す指標です。
definition: 相関係数は、2 つの変量がどれくらい直線的に一緒に増減するかを、単位の影響を取り除いて表す指標です。値は $-1$ から $1$ の間をとり、$1$ に近いほど強い正の直線関係、$-1$ に近いほど強い負の直線関係、0 に近いほど直線関係が弱いと読みます。ただし、相関係数は関係を説明的に要約する量であり、片方からもう片方を予測する式そのものではありません。
intuition:
  - 共分散は単位で大きさが変わりますが、相関係数は標準化しているので比較しやすいです。
  - 点が細く右上がりに並ぶ散布図なら、相関係数は 1 に近づきます。
  - 点が細く右下がりに並ぶ散布図なら、相関係数は -1 に近づきます。
  - 点が広く散っていれば、相関係数は 0 に近づきます。
  - 相関係数は「関係の向きと直線的な強さ」を要約する道具で、予測は回帰直線の仕事です。
visual_explanation:
  - 同じ右上がりでも、点が直線の近くに密集しているほど相関係数の絶対値は大きくなります。
  - 点が右上がりでも扇形に広がるなら、正の相関はあっても強くはありません。
  - U 字型の散布図では関係がありそうでも、直線的ではないため相関係数は 0 に近いことがあります。
where_it_appears:
  - 散布図を 1 つの数で要約して比較したいとき
  - 2 つの量の直線的な結びつきを簡潔に述べたいとき
  - 回帰直線や決定係数の理解に進む前に、説明的な関係の強さを整理するとき
practical_examples:
  - title: 勉強時間と得点
    description: 散布図が右上がりで点も比較的まとまっていれば、相関係数は正で大きめになります。これは勉強時間が長いほど得点が高い傾向を要約したもので、何点上がるかを直接予測する式ではありません。
  - title: 身長と体重
    description: 身長と体重には正の相関が見られやすいですが、相関が高くても個人差は残ります。相関係数は「平均的に一緒に増えやすい」ことを示すだけで、各人の体重をぴったり当てるものではありません。
  - title: U字型の失敗例
    description: 温度と売上が中間で最大になる U 字型の関係では、散布図では関係が見えるのに相関係数は 0 に近くなることがあります。相関係数は直線的な関係だけを見ています。
exam_points:
  - 相関係数の値は $-1$ から $1$ の範囲をとります。
  - 符号は関係の向き、絶対値は直線的な関係の強さの目安です。
  - 相関係数が大きくても、因果関係があるとは限りません。
  - 相関係数が 0 に近くても、直線でない関係が隠れていることがあります。
  - 相関係数は説明的な要約であり、予測に使うのは回帰直線です。
formulas:
  - name: 相関係数
    latex: 'r=\frac{s_{xy}}{s_x s_y}'
    description: 共分散 $s_{xy}$ を、$x$ と $y$ の標準偏差 $s_x$、$s_y$ で割って標準化したものが相関係数です。単位の違いを取り除いて、直線的な関係の強さを比べやすくします。
    conditions:
      - '$s_{xy}$ は $x$ と $y$ の共分散です。'
      - '$s_x$ と $s_y$ はそれぞれの標準偏差です。'
      - '$s_x>0$、$s_y>0$ を前提にします。'
  - name: 相関係数の範囲
    latex: '-1 \le r \le 1'
    description: 相関係数は必ず $-1$ 以上 $1$ 以下になります。端に近いほど直線的な並びが強く、0 に近いほど直線関係は弱いと読めます。
    conditions:
      - これは直線的な関係の強さについての範囲です。
rigorous_explanation:
  - 相関係数は、共分散を標準偏差で割って無次元化した量です。そのため、cm と kg のように単位が違う変量同士でも比較しやすくなります。
  - $r$ は $x$ と $y$ を対称に扱う量であり、$x$ と $y$ を入れ替えても値は変わりません。この点は、どちらからどちらを予測するかで式が変わる回帰と異なります。
  - 相関係数が表すのは直線的な関係の強さです。曲線的な関係や外れ値の影響は、散布図も合わせて確認しないと見誤ることがあります。
common_mistakes:
  - 相関係数を、因果関係の強さだと思うこと
  - 相関係数が高いなら、そのまま精度の高い予測ができると考えること
  - 相関係数が 0 なら、どんな関係もないと決めつけること
  - 散布図を見ずに、相関係数の数値だけで判断すること

相関係数は、散布図の形を「直線的な関係の強さ」という 1 つの数に要約したものです。`regression-line` はここからさらに進んで、説明変数 $x$ を決めたうえで目的変数 $y$ を予測する直線なので、対称な相関係数とは役割が違います。`no-correlation` もあわせて見ると、$r=0$ の読み方を誤解しにくくなります。
