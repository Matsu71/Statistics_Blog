title: 第二種の過誤

category: hypothesis_testing
short_definition: 実際には差や効果があるのに、帰無仮説を棄却できず見逃してしまう誤りです。
definition: 第二種の過誤とは、実際には対立仮説の側に差や効果があるのに、検定の結果として帰無仮説 $H_0$ を棄却できない誤りです。いわば「差があるのに差がないと判断してしまう」誤りで、偽陰性や false negative にあたります。第二種の過誤確率は通常 $\beta$ と書かれ、標本サイズ、ばらつき、真の差の大きさ、有意水準などで変わります。第二種の過誤が小さいほど、真の差を見つけやすくなります。
intuition:
  - 第二種の過誤は、本当は異常があるのに警報を鳴らし損ねる状態です。
  - 標本が小さい、ばらつきが大きい、差が小さいときに起こりやすくなります。
  - 帰無仮説を棄却できなかったからといって、差がないと確定したわけではありません。
  - 有意水準を厳しくすると、第一種の過誤は減りやすい一方で、第二種の過誤は増えやすくなります。
  - 第二種の過誤の裏返しが検出力です。
visual_explanation:
  - 対立仮説が真のときの分布を描き、その上で棄却域の外側に残る面積が第二種の過誤に関係します。
  - 真の差が小さいほど、その分布は帰無仮説の分布と重なりやすく、見逃しやすくなります。
  - 標本サイズを大きくすると分布のぶれが小さくなり、棄却域に届きやすくなります。
  - 第一種の過誤は帰無仮説側の誤り、第二種の過誤は対立仮説側の誤りと見ると整理しやすいです。
where_it_appears:
  - 効果があるのに検出できなかった状況を説明する場面
  - 検出力や標本サイズ設計の基本を考える場面
  - 第一種の過誤とのトレードオフを問う場面
  - 「棄却できない」を「正しい」と誤読しないための確認場面
practical_examples:
  - title: 効果がある薬を見逃す
    description: 実際には薬が効いているのに、標本サイズが小さくばらつきも大きいため p値が大きくなり、帰無仮説を棄却できなかったとします。これは第二種の過誤です。
  - title: 小さな改善を拾えない
    description: 工程改善で平均重量がわずかに変わっていても、標本数が少ないと検定統計量が十分大きくならず、異常を見逃すことがあります。差が小さいほど第二種の過誤は起こりやすくなります。
  - title: 標本サイズを増やす効果
    description: 同じ差でも標本サイズを増やすと標準誤差が小さくなり、検定統計量は大きくなりやすくなります。その結果、第二種の過誤は小さくなりやすいです。
exam_points:
  - 第二種の過誤は「差があるのに棄却できない」誤りです。
  - 第二種の過誤確率は通常 $\beta$ で表します。
  - 第二種の過誤は、標本サイズ、ばらつき、真の差の大きさ、有意水準に影響されます。
  - 検出力は $1-\beta$ で、真の差を見つける力を表します。
  - 帰無仮説を棄却できないことと、帰無仮説が真であることは同じではありません。
formulas:
  - name: 第二種の過誤確率
    latex: '\beta=P(T\notin R\mid H_1)'
    description: 実際には対立仮説の側にあるのに、検定統計量 $T$ が棄却域 $R$ に入らず、帰無仮説を棄却できない確率が第二種の過誤確率です。
    conditions:
      - '$R$ は棄却域です。'
      - '$H_1$ は対立仮説が真である状況を表します。'
      - 複数の真の値を含む対立仮説では、$\beta$ はどの真の値かによって変わります。
  - name: 検出力
    latex: '\mathrm{Power}=1-\beta'
    description: 真の差があるときに、それを正しく検出して帰無仮説を棄却できる確率を検出力といいます。
    conditions:
      - 第二種の過誤を小さくするほど、検出力は大きくなります。
      - 標本サイズ設計では、必要な検出力を先に決めることがあります。
rigorous_explanation:
  - 第二種の過誤は、対立仮説が真である側での見逃しです。したがって、その大きさは「どれくらいの差が本当にあるか」に依存します。
  - 同じ検定でも、真の差が大きければ検定統計量は棄却域へ届きやすくなり、第二種の過誤は小さくなります。差が小さいと見逃しやすくなります。
  - 標本サイズを大きくすると標準誤差が小さくなり、同じ差でも検出しやすくなるため、第二種の過誤は一般に小さくなりやすいです。
  - 第一種の過誤を抑えるために有意水準を下げすぎると、第二種の過誤は大きくなりやすいので、両者は一緒に考える必要があります。
proof:
  summary: 第二種の過誤は、対立仮説が真のときに観測統計量が棄却域へ届かないことで起こります。
  outline_steps:
    - 'まず、真の値が対立仮説の側にある状況を考えます。'
    - 'そのときの検定統計量 $T$ の分布を考えると、帰無仮説のときより分布の中心がずれることがあります。'
    - 'しかし、ばらつきや標本サイズの影響で、観測値が棄却域 $R$ に届かないこともあります。'
    - 'その確率 $P(T\notin R\mid H_1)$ が第二種の過誤確率 $\beta$ です。'
    - 'したがって、第二種の過誤は「真の差があるのに検定で拾えない確率」だと分かります。'
common_mistakes:
  - 第二種の過誤を「帰無仮説が真なのに棄却する誤り」と取り違えること
  - 帰無仮説を棄却できなかったら、帰無仮説が真だと断定すること
  - 第二種の過誤は常に $1-\alpha$ だと思うこと
  - 真の差の大きさや標本サイズが第二種の過誤に影響しないと思うこと

`type-i-error` と比べると、第二種の過誤は誤りの向きが逆です。第一種の過誤が「ない差をあると見てしまう誤り」なら、第二種の過誤は「ある差を見逃す誤り」です。

この違いを押さえると、検出力の意味も見えやすくなります。帰無仮説を棄却できなかった場面では、差がなかったのか、差があったのに検出力が足りなかったのかを区別して考える必要があります。
