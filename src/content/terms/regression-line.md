short_definition: 説明変数 $x$ から目的変数 $y$ を予測するときに、平均的な関係を直線で表したものです。
definition: 回帰直線は、散布図に見られる平均的な傾向を、$\hat{y}=a+bx$ という直線で表したものです。$x$ が 1 単位増えたときに、$y$ が平均的にどれくらい変わるかを読み取るのに使います。ただし、回帰直線は相関係数のような対称な要約ではなく、どちらを説明変数、どちらを目的変数とするかを決めて引く予測のための直線です。したがって、相関があることと、回帰で予測することは同じ意味ではありません。
intuition:
  - 散布図が右上がりや右下がりに並ぶとき、その真ん中を通る代表線として回帰直線を考えます。
  - 回帰直線は、各点をぴったり通す線ではなく、各 $x$ に対する平均的な $y$ の位置を表す線です。
  - 傾き $b$ が正なら $x$ が大きいほど $y$ も大きくなりやすく、負なら逆向きの傾向を表します。
  - 回帰直線では $x$ から $y$ を予測する向きが決まっているので、$x$ と $y$ を入れ替えると別の直線になります。
  - 相関係数が「一緒に動く強さ」を対称にまとめるのに対し、回帰直線は「$x$ から $y$ をどう見積もるか」を表します。
visual_explanation:
  - 散布図の点群の中央を、縦方向のずれが全体として小さくなるように通る直線だと考えるとイメージしやすいです。
  - 各点から回帰直線へ下ろした縦の距離が残差で、この距離が小さいほど直線のあてはまりはよいと読めます。
  - 最小二乗法で求めた回帰直線は、切片を含む単回帰では平均点 $(\bar{x},\bar{y})$ を通ります。
where_it_appears:
  - 勉強時間から得点、広告費から売上のように、1 つの量から別の量を平均的に予測したいとき
  - 散布図の形を、傾きと切片を持つ直線で要約したいとき
  - 単回帰分析、最小二乗法、決定係数を学ぶ入口として
practical_examples:
  - title: 勉強時間と得点の回帰直線
    description: 4 人の勉強時間を $x=(1,2,3,4)$ 時間、得点を $y=(52,54,61,63)$ 点とすると、最小二乗法で求めた回帰直線は $\hat{y}=47.5+4x$ です。勉強時間が 1 時間増えると、得点は平均的に 4 点上がる向きだと読みます。
  - title: 同じ相関でも回帰直線は向きで変わる
    description: 身長から体重を予測する直線と、体重から身長を予測する直線は別物です。相関係数は同じでも、回帰直線はどちらを説明変数に置くかで変わります。
  - title: 切片の読み方
    description: 勉強時間が 0 時間のときの予測得点が切片ですが、実際に 0 時間という状況が観測範囲にないなら、切片を現実にそのまま解釈しすぎないようにします。
exam_points:
  - 回帰直線は、説明変数 $x$ から目的変数 $y$ を予測するための直線です。
  - 傾き $b$ は、$x$ が 1 単位増えたときに $y$ が平均的にどれだけ変化するかを表します。
  - 相関係数は対称な要約ですが、回帰直線は向きがあるので $x$ と $y$ を入れ替えると変わります。
  - 回帰直線がよくあてはまっても、それだけで因果関係が示されたとはいえません。
  - 観測した範囲の外まで直線をそのまま延長して予測する外挿には注意が必要です。
formulas:
  - name: 回帰直線
    latex: '\hat{y}=a+bx'
    description: 説明変数 $x$ から、目的変数 $y$ の予測値 $\hat{y}$ を直線で表した式です。$a$ は切片、$b$ は傾きです。
    conditions:
      - '$\hat{y}$ は予測値です。'
      - '$a$ は $x=0$ のときの予測値を与える切片です。'
      - '$b$ は $x$ が 1 単位増えたときの平均的な変化量です。'
  - name: 傾き
    latex: 'b=\frac{s_{xy}}{s_x^2}'
    description: 傾きは、$x$ と $y$ の共分散 $s_{xy}$ を、$x$ の分散 $s_x^2$ で割って求められます。$x$ のばらつきに対して、$y$ がどちら向きにどれだけ動くかを表します。
    conditions:
      - '$s_{xy}$ は $x$ と $y$ の共分散です。'
      - '$s_x^2$ は $x$ の分散で、$s_x^2>0$ を前提にします。'
      - 切片を含む単回帰で、最小二乗法を使うとこの形になります。
  - name: 切片
    latex: 'a=\bar{y}-b\bar{x}'
    description: 回帰直線が平均点 $(\bar{x},\bar{y})$ を通ることから、切片は平均と傾きで決まります。
    conditions:
      - '$\bar{x}$ は $x$ の平均、$\bar{y}$ は $y$ の平均です。'
      - 切片を含む単回帰を考えています。
rigorous_explanation:
  - 回帰直線は、$x$ から $y$ を予測する向きが決まった直線です。この「向き」がない相関係数とは役割が異なります。
  - 最小二乗法で回帰直線を求めると、各観測値 $y_i$ と予測値 $\hat{y}_i$ の縦方向のずれ $e_i=y_i-\hat{y}_i$ の二乗和が最小になります。
  - 傾き $b=\frac{s_{xy}}{s_x^2}$ は、$x$ のばらつき 1 単位あたりに $y$ がどれだけ連動しているかを見る形です。共分散の符号が、回帰直線の向きにも反映されます。
  - 切片を含む単回帰では、回帰直線は平均点 $(\bar{x},\bar{y})$ を通ります。したがって、平均のまわりでの傾向を直線でまとめていると考えられます。
  - 回帰直線は予測や要約の道具であって、直線が引けたこと自体から因果関係や機構まで断定することはできません。
proof:
  summary: 最小二乗法で残差平方和を最小にすると、回帰直線の傾きと切片の公式が得られます。
  outline_steps:
    - '残差平方和 $S(a,b)=\sum_{i=1}^{n}(y_i-a-bx_i)^2$ を考えます。'
    - '$a$ と $b$ で偏微分して 0 とおくと、$\sum_{i=1}^{n}(y_i-a-bx_i)=0$ と $\sum_{i=1}^{n}x_i(y_i-a-bx_i)=0$ が得られます。'
    - '第 1 式から $a=\bar{y}-b\bar{x}$ が分かります。'
    - 'これを第 2 式へ代入して整理すると、$b=\frac{s_{xy}}{s_x^2}$ が得られます。'
    - 'したがって、最小二乗法で求めた回帰直線は $\hat{y}=a+bx$ の形で一意に決まります。'
common_mistakes:
  - 回帰直線と相関係数を同じものだと思うこと
  - 相関が高ければ、$x$ と $y$ を入れ替えても同じ回帰直線になると考えること
  - 傾きを、そのまま因果効果の大きさだと決めつけること
  - 観測範囲から大きく外れた $x$ にまで、回帰直線を無条件に延長して使うこと

`simple-linear-regression` は、回帰直線を誤差項まで含めたモデルとして扱うページです。実際にその直線をどう決めるかは `least-squares-method`、どれくらいデータの変動を直線で表せているかは `coefficient-of-determination` につながります。

相関係数は「一緒に動く強さ」を対称に要約する量ですが、回帰直線は「どちらからどちらを予測するか」を決めたうえで引く直線です。この違いを押さえておくと、相関と回帰を混同しにくくなります。
