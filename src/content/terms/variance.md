title: 分散
slug: variance
exam_scope:
  - statistics_grade_2
level: standard

category: descriptive_statistics
short_definition: データが平均のまわりにどのくらい散らばっているかを、二乗を使って測る代表的な指標です。
definition: 各データの平均からのずれを二乗し、その平均をとった値を分散といいます。平均が同じでも、広がり方が違うデータを区別するために使います。分散は、中心だけでは分からないデータのばらつきを数で表すための基本です。ただし、単位は元データの二乗になるため、値の大きさを読むときは標準偏差とあわせて考えます。
intuition:
  - 平均が同じ 2 つのデータでも、値が平均の近くに集まる場合と大きく離れる場合があります。
  - 分散は、その「平均からの離れやすさ」をひとまとめにした数字です。
  - 偏差をそのまま足すと正負が打ち消し合うので、二乗してから平均するという工夫をしています。
  - 大きい値ほど強く効くので、外れた値があると分散は大きくなりやすいです。
  - 分散は中心の位置ではなく、中心のまわりにどれくらい広がっているかを表します。
  - 範囲が端の 2 点を見るのに対して、分散はすべての値が平均からどれくらい離れているかを使います。
visual_explanation:
  - 平均が同じでも、点が狭く集まる集団は分散が小さく、広く散る集団は分散が大きくなります。
  - 平均から遠い値ほど二乗の効果で強く効くため、外れた値があると分散は大きくなりやすいです。
  - 平均のまわりに点を並べたとき、左右に広がる幅が大きいほど分散も大きいと考えられます。
  - 平均から各点までの距離を見て、それを二乗してからならす、と考えると式の意味が追いやすくなります。
where_it_appears:
  - 平均だけでは見えないデータの広がり方を比べたいときに使います。
  - 標準偏差、正規分布、推定、検定などを学ぶ前提として繰り返し登場します。
  - 品質管理やテスト結果の安定性の比較のように、ばらつき自体が重要な場面で現れます。
  - 標本平均の散らばりを考えるときにも土台になります。
practical_examples:
  - title: テストの得点分布
    description: 平均点がどちらも 70 点でも、60 点から 80 点に集中しているクラスと、30 点から 100 点まで広がっているクラスでは分散が違います。後者のほうが分散は大きいです。
  - title: '例題: 同じ平均で分散を比べる'
    description: 60, 70, 80 と 50, 70, 90 はどちらも平均 70 ですが、後者のほうが平均から 20 点離れた値を含むため分散が大きくなります。平均が同じでも、散らばりは別に確認します。
  - title: 工場での測定値
    description: 製品の長さや重さが平均値の近くで安定しているかを見るときに、分散や標準偏差が使われます。安定していれば分散は小さくなります。
  - title: 待ち時間の比較
    description: 2 つの窓口で平均待ち時間が同じでも、ある日は 5 分と 20 分が混ざるより、8 分から 12 分に収まるほうが分散は小さくなります。
  - title: 範囲との違い
    description: 範囲は最大値と最小値だけを見ますが、分散はすべての観測値を使います。端の幅だけでは分からない、平均まわりの散らばり方を補って読む指標です。
exam_points:
  - 分散は散らばりの大きさを見る基本指標です。
  - 分散の単位は元データの単位ではなく、その二乗になります。
  - 標準偏差は分散の平方根であり、単位を元に戻した指標です。
  - 分散は平均からのずれを二乗しているので、外れ値の影響を受けやすいです。
  - 問題文では、平均の違いと散らばりの違いを分けて読むことが大切です。
  - このページの $s_n^2$ は、観測値の個数 $n$ で割る「データの分散」として扱っています。不偏分散とは区別します。
formulas:
  - name: データの分散
    latex: 's_n^2 = \frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})^2'
    description: 平均からのずれを二乗し、その平均をとると分散になります。データ全体の広がりを 1 つの値にまとめた形です。ここでは観測値 $n$ 個で割る分散を表しています。
    conditions:
      - '$x_i$ は i 番目の観測値です。'
      - '$\bar{x}$ は平均です。'
      - 平均から遠い値ほど、二乗によって寄与が大きくなります。
      - 不偏分散ではなく、手元のデータの散らばりを $n$ でならす式です。
  - name: 分散の展開形
    latex: 's_n^2 = \frac{1}{n}\sum_{i=1}^{n}x_i^2-\bar{x}^2'
    description: 平均からのずれを二乗する式は、二乗の平均から平均の二乗を引く形にも整理できます。計算式として使うより、分散が「二乗した大きさ」と「中心」の差を見ていると読むための形です。
    conditions:
      - '$\frac{1}{n}\sum_{i=1}^{n}x_i^2$ は観測値の二乗の平均です。'
      - '$\bar{x}^2$ は平均の二乗です。'
      - 同じ $n$ 個の観測値についての恒等式です。
  - name: 分散と標準偏差の関係
    latex: '\sigma = \sqrt{\sigma^2}'
    description: 分散の平方根をとると標準偏差になります。単位を元のデータに戻して解釈しやすくしたものです。
    conditions:
      - 分散と標準偏差は役割が近いですが、単位が異なります。
rigorous_explanation:
  - 偏差の和 $\sum_{i=1}^{n}(x_i-\bar{x})$ は 0 になるので、散らばりの大きさをそのまま測るには使いにくいです。そのため、負の値も正にし、遠い値を強く反映する二乗を使います。
  - 分散は「平均からの二乗距離の平均」とみることができます。この見方を持つと、標本分散や不偏分散の違いも整理しやすくなります。
  - このページの分散は、観測値 $n$ 個を 1 つのデータ集合として見たときの散らばりです。推測統計で使う不偏分散では、割る数が $n-1$ になる場面があります。
  - 分散が 0 になるのは、すべての観測値が平均と等しいときです。どれか 1 つでも平均から離れていれば、二乗したずれが正になり、分散も正になります。
  - 標本平均が何回か変わると、そのぶれの大きさを考える場面にも分散の考え方が出てきます。
proof:
  summary: 偏差の和が 0 になることを確認すると、なぜ単純な偏差の平均では散らばりを測れず、二乗が必要になるかが分かります。
  outline_steps:
    - '偏差 $x_i-\bar{x}$ は、各値が平均からどれだけ離れているかを表します。'
    - '偏差の和をとると、$\sum_{i=1}^{n}(x_i-\bar{x}) = \sum_{i=1}^{n}x_i - n\bar{x}$ です。'
    - '平均の定義から $n\bar{x}=\sum_{i=1}^{n}x_i$ なので、偏差の和は 0 になります。'
    - 'そのまま平均すると正のずれと負のずれが打ち消されるため、散らばりの大きさを表せません。'
    - 'そこで $(x_i-\bar{x})^2$ と二乗してから平均し、平均から遠い値ほど大きく反映させます。'
    - 'この考え方をまとめたものが $s_n^2 = \frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})^2$ です。'
common_mistakes:
  - 分散の値を、元データと同じ単位でそのまま解釈してしまうこと
  - 分散が大きいことを、平均が大きいことと混同すること
  - 標準偏差と分散を同じものとして扱い、平方根の関係を説明できないこと
  - 分散は小さいほどデータが必ず同じになると考えてしまうこと
  - $n$ で割る分散と $n-1$ で割る不偏分散を、説明なしに同じものとして扱うこと

分散は、平均だけでは見えない「広がり方」を言葉と式の両方でつかむための入口です。ここで二乗する理由まで納得しておくと、`standard-deviation` や `normal-distribution`、さらに推定や検定へ進んだときに理解がつながりやすくなります。

`range` が端の 2 点だけを見るのに対して、分散はすべての値と平均との距離を使います。`standard-deviation` はこの分散の平方根をとり、元の単位で散らばりを読みやすくしたものです。
